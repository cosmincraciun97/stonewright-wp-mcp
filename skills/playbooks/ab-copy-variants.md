# A/B copy variants

## Steps
1. Read current hero/CTA copy from the live page structure.
2. Draft 2–3 variants preserving brand voice from site memory/skills.
3. Apply the chosen variant only after user selection.
4. Snapshot before write; mutate text nodes only.

## Rules
- Do not invent brand claims; vary framing and urgency, not facts.
