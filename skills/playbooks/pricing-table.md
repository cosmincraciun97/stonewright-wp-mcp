# Pricing table

## Steps
1. Collect tier names, prices, and feature lists from the user.
2. Spec 2–4 columns; highlight recommended plan.
3. Validate + build; keep currency/period exact.
4. CTAs link to real checkout/contact targets.

## Rules
- No invented discounts or “most popular” without user direction.
