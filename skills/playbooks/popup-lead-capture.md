# Popup lead capture

## Steps
1. Discover form shortcodes with `stonewright/site-shortcodes-discover`.
2. Prefer builder-native popup/template tools when Elementor Pro is present (`theme-builder-apply-template`).
3. Keep copy short; one form, one CTA.
4. Snapshot templates before write.

## Rules
- Do not inject third-party scripts without user request.
