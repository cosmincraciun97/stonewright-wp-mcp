# Homepage hero

## Steps
1. Read current front page via `stonewright/site-info` / content get.
2. Source hero image: library first, else stock import with attribution.
3. Spec hero with max two CTAs; validate; apply via builder tools.
4. Set alt text with `stonewright/media-set-alt`.

## Rules
- Prefer full-bleed controlled height over random tall images.
