# Stonewright 1.0.0-alpha.8

Release date: 2026-06-11

This alpha clarifies Stonewright's Composer dependency audit policy for the
WordPress Abilities API compatibility package.

## Highlights

- Keeps `wordpress/abilities-api` as the compatibility package for supported
  WordPress versions that do not yet ship the Abilities API in core.
- Configures Composer audit to report abandoned packages without failing
  releases when there are zero security vulnerability advisories.
- Adds a dedicated `composer dependencies:audit` release check.
- Documents when the compatibility package can be removed: after Stonewright's
  supported WordPress floor includes the core Abilities API.

## Release Assets

- `stonewright-1.0.0-alpha.8.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.8.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
