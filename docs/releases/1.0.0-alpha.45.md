# Stonewright 1.0.0-alpha.45

Released: 2026-06-16

## Highlights

- Makes compact companion profile selection more forgiving so agents can use
  natural task names without accidentally falling back to the generic
  `essential` tool surface.

## Changed

- `STONEWRIGHT_MCP_TOOL_PROFILE` now accepts aliases such as `elementor`,
  `design`, `acf`, `cpt-ui`, `fse`, and `wp cli`, then normalizes them to the
  matching compact profile before proxied WordPress MCP tools are filtered.
- `stonewright-setup-profile` now documents the alias behavior in its compact
  setup notes.

## Verification

- `npm test -- mcp-server setup-profile`

## Assets

- `stonewright-1.0.0-alpha.45.zip`: installable WordPress plugin package.
- `stonewright-companion-1.0.0-alpha.45.tgz`: Node companion package.
