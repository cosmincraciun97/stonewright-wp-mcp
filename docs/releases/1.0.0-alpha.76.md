# 1.0.0-alpha.76

## Highlights

- **Permanent Direct HARD RULES:** every `stonewright-task-start` returns
  product-default operating rules (single-target environment, remote Direct
  tool path, no ad-hoc plugins, HTTP-first automation, additive content models).
- **New Direct builtin skills** (enabled by default): no-ad-hoc-plugins,
  single-environment-scope, remote-direct-tool-path, http-first-admin,
  content-model-additive.
- **Plugin permanent rules** injected into agent instructions — **not** stored
  as Safety Memory UI rows. Includes Elementor native-first lessons (responsive
  typography, Nested Carousel, swiper overflow, CSS parent class).
- **Skill pack** `agent-operating-rules` for on-demand playbook matching.

## Upgrade notes

- Update companion and plugin together.
- Direct: restart/reload the MCP client so new builtins seed under
  `~/.stonewright/skills/_builtin/` (copy-if-missing; existing edits preserved).
- Plugin: permanent rules apply on next `stonewright-task-start` without importing
  knowledge bundles into Safety Memory.
- After upgrade, call `stonewright-task-start` and confirm guidance includes
  `HARD RULE` lines (Direct) or permanent operating rules (plugin).
