# Stonewright 1.0.0-alpha.29

Released: 2026-06-16

## Highlights

- Tool profiles now expose recovery data when a profile expects a tool that is
  disabled, gated, or outside the current compact surface.
- Agents can avoid slow fallback loops by inspecting `missing_profile_tools`,
  `missing_mcp_tools`, and `recovery_hints` before continuing an Elementor,
  Gutenberg, content-model, or WP-CLI workflow.

## Changed

- `stonewright/tool-profile` filters `recommended_tools` to enabled abilities
  only.
- Added compact missing-tool metadata and counts to the profile response.
