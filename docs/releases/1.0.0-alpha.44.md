# Stonewright 1.0.0-alpha.44

Released: 2026-06-16

## Highlights

- Adds compact startup-readiness diagnostics to the companion WordPress MCP
  status tool so agents can recover from missing first-call tools without broad
  discovery.

## Changed

- `stonewright-wordpress-mcp-status` now reports `startup_ready`,
  `startup_required_tool_names`, and `startup_missing_tool_names`.
- Status output now includes `local_recovery_tool_names`, listing the companion
  tools that remain available while the WordPress MCP endpoint or startup tool
  surface is repaired.
- Recovery hints now tell agents to update or enable the missing Stonewright
  startup tools when `startup_ready` is false.

## Verification

- `npm test -- mcp-server`

## Assets

- `stonewright-1.0.0-alpha.44.zip`: installable WordPress plugin package.
- `stonewright-companion-1.0.0-alpha.44.tgz`: Node companion package.
