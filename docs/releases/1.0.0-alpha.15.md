# Stonewright 1.0.0-alpha.15

Release date: 2026-06-15

This alpha makes Stonewright faster for real WordPress building work while
keeping the same control model: permission checks, backups, validation,
diagnostics, and production-safe confirmations.

## Highlights

- Adds Fast Apply v1 for Elementor through
  `stonewright/elementor-v3-batch-mutate`, so agents can add, update, move, and
  remove many elements on a page with one compact write.
- Adds `stonewright/content-bulk-upsert-posts` for CPT, post, and custom-field
  data seeding in one request instead of many repeated WP-CLI/meta commands.
- Extends `stonewright/elementor-v3-build-page-from-spec` with dry runs,
  timing metrics, diagnostics, element counts, and `replace`, `append`, and
  `replace_section` modes.
- Adds essential tools mode for a compact high-value tool set that reduces MCP
  discovery noise and token use.
- Adds `stonewright-setup-profile` to the companion for intuitive
  cross-platform setup on Windows, macOS, and Linux.

## Release Assets

- `stonewright-1.0.0-alpha.15.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.15.tgz`: Node companion package for
  WP-CLI, batch WP-CLI, health checks, optional MCP proxy, setup profiles, and
  Stonewright MCP prompts.
- `SHA256SUMS.txt`: checksums for release assets.
