# Stonewright 1.0.0-alpha.31

Released: 2026-06-16

## Highlights

- Empty input schemas now serialize `properties` as `{}` instead of `[]`, which
  keeps stricter MCP clients from rejecting or hiding no-input tools.
- Tool discovery is a bit less fragile for new sessions, so agents spend fewer
  retries and less context recovering from schema-shape mismatches.

## Changed

- Normalized JSON Schema object-map keys centrally in `AbilityRegistry`.
- Added unit coverage for no-input ability schema encoding.
