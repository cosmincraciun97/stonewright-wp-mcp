# Stonewright 1.0.0-alpha.34

Released: 2026-06-16

## Highlights

- Fixed compact companion proxy profiles so new MCP sessions can actually see
  `stonewright-media-list` when PHP-side Stonewright profiles recommend media
  discovery before upload.

## Fixed

- Essential, Elementor design, and content-model companion profiles now include
  `stonewright-media-list` alongside `stonewright-media-upload-batch`.
- Essential companion proxy discovery no longer exposes single
  `stonewright-media-upload` as the compact path.
