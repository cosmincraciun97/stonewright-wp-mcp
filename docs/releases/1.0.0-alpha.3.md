# Stonewright 1.0.0-alpha.3

Release date: 2026-06-11

This alpha fixes the CI failures seen on `main`, improves first-pass MCP
workflow speed, and refreshes release packaging for public downloads.

## Highlights

- Added `stonewright/workflow-preflight` for compact one-call task setup.
- Added `stonewright/media-upload-batch` for per-item batch uploads.
- Added `stonewright/elementor-v3-capabilities-summary` and
  `stonewright/elementor-v3-apply-bundle` for faster guarded Elementor work.
- Fixed Composer dependency resolution on PHP 8.1 by pinning the Composer
  platform and refreshing the lockfile.
- Fixed WP-CLI discovery so LocalWP phars near the site root win over the
  generic companion cache.
- Added Elementor fixed-width row overflow diagnostics.
- Added release automation for plugin ZIP, companion TGZ, and checksums.
- Expanded release-download installation docs and client connection snippets.

## Release Assets

- `stonewright-1.0.0-alpha.3.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.3.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
