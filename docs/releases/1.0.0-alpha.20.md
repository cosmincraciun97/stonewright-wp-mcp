# Stonewright 1.0.0-alpha.20

Release date: 2026-06-16

## Highlights

- Adds `stonewright/tool-profile`, a compact task-aware profile tool for
  Elementor design, content-model, Gutenberg, WP-CLI, site-admin, and full
  discovery workflows.
- Returns `tool_profile_hint` from `stonewright/context-bootstrap` and adds
  `stonewright/tool-profile` to workflow preflight call sequences so agents can
  stay token-efficient before broad discovery.
- Improves Elementor V3/V4 status with `v4_write_ready`,
  `recommended_renderer`, and `agent_action` so agents avoid V4 writes when
  atomic support is unavailable or disabled.
