# 1.0.0-alpha.67

Premium admin shell + Direct mode + learning loop.

## Highlights

- Dark/light admin UI on semantic tokens (usable dark mode).
- Blueprints page cards, Setup HTTP as info, Abilities/Sandbox layout fixes.
- Companion Direct mode for plugin-less WordPress REST work (HTTP OK).
- Feedback capture, audit recurring-error learning, task-start warnings.
- Token surface budgets in CI.

## Abilities

- Added: `stonewright/feedback-capture`
- Extended: `stonewright/learning-record` (`trigger`, `severity`, `source`)
- Context: `custom_instructions`, `recurring_errors` on bootstrap/task-start

## Security gates

Unchanged: Permissions, Backup, Validator, ConfirmationToken, AuditLog (AuditLog
now also feeds ErrorPatterns best-effort after ERROR rows).

## Verify

```bash
cd plugin && composer test && composer phpstan && composer phpcs
cd ../companion && npm run typecheck && npm test && npm run build
```
