# Stonewright 1.0.0-alpha.6

Release date: 2026-06-11

This alpha adds plugin-specific training for common WordPress content-model and
commerce workflows while keeping Stonewright's security envelope unchanged.

## Highlights

- Adds official-docs-based specialization guidance for ACF, ACPT, Meta Box,
  ASE, Pods, and WooCommerce catalog work.
- Adds built-in content-model and WooCommerce skills for agents that need fast,
  repeatable planning before editing fields, post types, taxonomies, products,
  or variations.
- Extends `stonewright/context-bootstrap` and `stonewright/workflow-preflight`
  with compact specialization summaries to reduce repeated discovery calls.
- Fixes sandbox admin test isolation so full PHPUnit runs stay deterministic.

## Release Assets

- `stonewright-1.0.0-alpha.6.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.6.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
