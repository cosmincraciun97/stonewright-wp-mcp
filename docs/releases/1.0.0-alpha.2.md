# Stonewright 1.0.0-alpha.2

Release date: 2026-06-11

This first public alpha prepares Stonewright for open-source evaluation and
local WordPress automation. It expands the Elementor-first building
surface, hardens write paths, improves the admin experience, and documents the
expected onboarding flow for MCP clients.

## Highlights

- Added public project hygiene: CI, issue templates, pull request template,
  code of conduct, contributing guide, security policy, onboarding guide, and
  open-source readiness checklist.
- Added a REST ability runner for authenticated local clients while preserving
  Stonewright permission callbacks and context-token gates.
- Expanded Elementor V3 editing and Elementor V4 experimental helpers.
- Added Elementor Widget Builder primitives.
- Added deeper Gutenberg and Full Site Editing operations.
- Added memory, system, sandbox, site, and content management abilities.
- Added confirmation tokens, pre-write backups, Design Spec validation,
  theme.json validation, StaticGuard, and audit logging.
- Improved admin pages for configuration, abilities, skills, memory, sandbox,
  and status.
- Added native Elementor layout guidance for global styles, responsive
  breakpoints, lazy-loaded media verification, and no-HTML-widget design builds.

## Release Assets

- `stonewright-1.0.0-alpha.2.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.2.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
