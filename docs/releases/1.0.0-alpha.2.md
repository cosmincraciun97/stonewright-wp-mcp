# Stonewright 1.0.0-alpha.2

Release date: 2026-05-22

This milestone expanded the Elementor-first WordPress building surface and
hardened the security envelope around writes.

## Highlights

- Expanded Elementor V3 editing and Elementor V4 experimental helpers.
- Added Elementor Widget Builder primitives.
- Added deeper Gutenberg and Full Site Editing operations.
- Added memory, system, sandbox, site, and content management abilities.
- Added confirmation tokens, pre-write backups, Design Spec validation,
  theme.json validation, StaticGuard, and audit logging.

## Current Follow-Up

The unreleased branch after this tag removes Stonewright-owned design-tool
ingestion and automated visual QA, then replaces that companion surface with
guarded WP-CLI support and mandatory persistent task context.
