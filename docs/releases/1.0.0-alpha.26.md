# Stonewright 1.0.0-alpha.26

Release date: 2026-06-16

## Highlights

- Companion proxied WordPress MCP tools now default to the compact `essential`
  profile when the env profile is unset.
- Reduces first-session MCP tool surface and token cost for clients that omit
  the generated setup-profile environment block.
- Keeps `STONEWRIGHT_MCP_TOOL_PROFILE=full`, `all`, `off`, `false`, and `0` as
  explicit opt-ins for sessions that need every proxied WordPress MCP tool.
