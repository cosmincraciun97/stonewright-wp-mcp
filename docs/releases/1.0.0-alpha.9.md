# Stonewright 1.0.0-alpha.9

Release date: 2026-06-11

This alpha improves Stonewright's low-token workflow setup for agents while
keeping the write control envelope unchanged.

## Highlights

- Adds task-aware `stonewright/workflow-preflight` profiling for surface,
  intent, write/destructive status, visual-check needs, WP-CLI discovery needs,
  context-token needs, and production-safe confirmation-token needs.
- Adds `recommended_mcp_tools` so MCP clients can use the exact hyphenated tool
  names without another discovery pass.
- Adds a compact `call_sequence` with example arguments for Elementor native
  widget planning, batch asset upload, spec-first Elementor writes, plugin/WP-CLI
  discovery, and production-safe destructive confirmation tokens.
- Keeps write gates unchanged: write abilities still require
  permission callbacks, context tokens, backups where required, validators, and
  confirmation tokens in production-safe mode.
- Updates public docs to describe the expanded preflight response contract.

## Release Assets

- `stonewright-1.0.0-alpha.9.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.9.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
