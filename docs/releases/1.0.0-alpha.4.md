# Stonewright 1.0.0-alpha.4

Release date: 2026-06-11

This alpha follows 1.0.0-alpha.3 with a PHP 8.1 compatibility fix for the
public CI matrix.

## Highlights

- Restored PHP 8.1 syntax compatibility in sandbox, security, companion, and
  test support signatures.
- Keeps the 1.0.0-alpha.3 workflow-speed abilities, guarded Elementor bundle
  writes, media batch uploads, release packaging, and installation docs.

## Release Assets

- `stonewright-1.0.0-alpha.4.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.4.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
