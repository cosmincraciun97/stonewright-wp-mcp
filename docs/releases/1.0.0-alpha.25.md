# Stonewright 1.0.0-alpha.25

Release date: 2026-06-16

## Highlights

- Extends companion startup resilience to WordPress MCP config resolution
  failures, including malformed site URLs before proxy discovery starts.
- Keeps local setup-profile, `stonewright-wordpress-mcp-status`, and direct
  `stonewright-wp-cli-*` tools visible while the user fixes connection config.
- Preserves compact startup and release-tarball setup from previous alpha
  releases.
