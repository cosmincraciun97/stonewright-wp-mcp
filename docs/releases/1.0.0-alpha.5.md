# Stonewright 1.0.0-alpha.5

Release date: 2026-06-11

This alpha tightens the visual-build workflow so agents do not build Elementor
pages blind when browser verification is unavailable.

## Highlights

- Adds a hard pre-write Playwright/browser MCP gate to task bootstrap,
  workflow preflight, and default agent instructions.
- Expands client setup docs and admin snippets with Playwright MCP install,
  restart, and tool-list verification steps.
- Adds Antigravity-specific onboarding guidance to configure both Stonewright
  and Playwright before visual work.
- Adds fast-path spec and bundle guidance so repeated cards/grids are built in
  one first pass, with individual Elementor calls reserved for screenshot
  corrections.

## Release Assets

- `stonewright-1.0.0-alpha.5.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.5.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
