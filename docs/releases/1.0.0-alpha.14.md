# Stonewright 1.0.0-alpha.14

Release date: 2026-06-12

This alpha makes repeated WP-CLI writes safer for agents and clarifies local
companion onboarding.

## Highlights

- Adds a UTF-8 safe WP-CLI batch runner in the companion, exposed as
  `stonewright-wp-cli-batch-run`, `companion_wp_cli_batch_run`, and
  `POST /wp-cli/batch`.
- Keeps repeated writes as argv arrays instead of large inline PowerShell or
  shell scripts, preventing mojibake when agents seed media, CPT, ACF,
  taxonomies, or posts with non-ASCII content.
- Documents that `STONEWRIGHT_WP_ROOT` is optional and should point to the
  absolute WordPress install folder containing `wp-config.php`.

## Release Assets

- `stonewright-1.0.0-alpha.14.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.14.tgz`: Node companion package for
  WP-CLI, batch WP-CLI, health checks, optional MCP proxy, and Stonewright MCP
  prompts.
- `SHA256SUMS.txt`: checksums for release assets.
