# Stonewright 1.0.0-alpha.47

Released: 2026-06-16

## Highlights

- Adds selected-profile missing tool diagnostics so agents can tell when the
  compact proxy profile is connected but a recommended non-startup Stonewright
  tool is absent.

## Changed

- `stonewright-wordpress-mcp-status` now includes
  `profile_expected_tool_count` and `profile_missing_tool_names`.
- Profile missing checks count local companion WP-CLI tools as available, so
  direct `stonewright-wp-cli-*` tools are not incorrectly reported missing.
- Recovery hints now point agents at enabling or updating missing profile tools,
  with `STONEWRIGHT_MCP_TOOL_PROFILE=full` as specialist recovery.

## Verification

- `npm test -- mcp-server`

## Assets

- `stonewright-1.0.0-alpha.47.zip`: installable WordPress plugin package.
- `stonewright-companion-1.0.0-alpha.47.tgz`: Node companion package.
