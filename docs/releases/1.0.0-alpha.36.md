# Stonewright 1.0.0-alpha.36

Released: 2026-06-16

## Highlights

- Hardened fast Elementor container writes so guessed flex controls from agents
  do not create overflow, wrapped rows, or unstable section sizing.
- Shared container settings normalization across add, batch, and update flows,
  keeping section-by-section design implementation consistent when agents mix
  fast batch writes with targeted Elementor fixes.

## Fixed

- `stonewright/elementor-v3-add-container` now strips risky container flex
  settings before writing Elementor data.
- `stonewright/elementor-v3-batch-mutate` now normalizes container settings on
  both `add_container` and `update_element` operations.
- `stonewright/elementor-v3-update-element` now applies the same container-aware
  normalization path when updating existing Elementor containers.
