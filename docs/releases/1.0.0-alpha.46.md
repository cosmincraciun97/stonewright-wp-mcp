# Stonewright 1.0.0-alpha.46

Released: 2026-06-16

## Highlights

- Makes companion recovery more explicit by listing every local setup and
  WP-CLI tool that remains available when proxied WordPress MCP tools are down
  or profile-filtered.

## Changed

- `stonewright-wordpress-mcp-status` now includes `local_tool_names` with setup,
  proxy status, Stonewright WP-CLI tools, and legacy companion WP-CLI aliases.
- `local_recovery_tool_names` now includes `stonewright-wp-cli-run` and
  `stonewright-wp-cli-install`.
- `stonewright-setup-profile.tool_visibility_checks` now asks agents to verify
  direct WP-CLI run and install tools too.

## Verification

- `npm test -- mcp-server setup-profile`

## Assets

- `stonewright-1.0.0-alpha.46.zip`: installable WordPress plugin package.
- `stonewright-companion-1.0.0-alpha.46.tgz`: Node companion package.
