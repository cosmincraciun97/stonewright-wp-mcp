# Stonewright 1.0.0-alpha.39

Released: 2026-06-16

## Highlights

- Preserved compact Elementor widget validation details in public error
  envelopes so agents can self-correct missing required settings without
  rediscovering tools or repeating failed calls.

## Fixed

- `ErrorEnvelope` now forwards sanitized `widget` and `violations` fields from
  repairable validation errors.
- Public error envelopes still strip sensitive or heavy fields such as tokens,
  passwords, raw specs, and non-allowlisted data.
