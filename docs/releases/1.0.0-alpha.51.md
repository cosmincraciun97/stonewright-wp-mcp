# Stonewright 1.0.0-alpha.51

Released: 2026-06-17

## Highlights

- Adds a compact tool inventory to companion setup and status responses so agents can pick the right Stonewright tools without broad discovery loops.
- Improves startup clarity for Codex, Claude, Gemini, Antigravity, DeepSeek, and other MCP clients that benefit from grouped tool guidance.

## Added

- `tool_inventory` in `stonewright-setup-profile`.
- `tool_inventory` in `stonewright-wordpress-mcp-status`.
- Grouped first-call, diagnostic, direct WP-CLI, long-running WP-CLI, and proxied profile tool names.
