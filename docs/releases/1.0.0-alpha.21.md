# Stonewright 1.0.0-alpha.21

Release date: 2026-06-16

## Highlights

- Extends `stonewright-setup-profile` with `first_calls` and
  `tool_visibility_checks`, giving new MCP sessions a compact checklist for
  bootstrap, preflight, tool profile, and direct WP-CLI aliases.
- Updates the WordPress admin copy-paste setup prompt and client docs so agents
  call `stonewright-tool-profile` for tool-cap, slow-startup, or
  token-sensitive clients before broad discovery.
