# Stonewright 1.0.0-alpha.35

Released: 2026-06-16

## Highlights

- Tightened MCP schema compatibility for stricter clients by ensuring public
  array input parameters declare `items: {}` when they are intentionally
  permissive.
- Updated Elementor widget intent prerequisites so media-based intents search
  existing WordPress media first, then batch-upload only missing assets.

## Fixed

- `stonewright/blocks-serialize`, `stonewright/blocks-insert`, and
  `stonewright/design-build-spec` no longer expose array input parameters
  without an `items` schema after central registry normalization.
- Media-related widget intents no longer recommend the slower
  `stonewright/media-upload` single-call path.
