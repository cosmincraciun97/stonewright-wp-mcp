# Stonewright 1.0.0-alpha.40

Released: 2026-06-16

## Highlights

- Tightened companion compact proxy profiles so Elementor design sessions see
  the same container-schema fast path that Stonewright PHP preflight recommends.

## Fixed

- `STONEWRIGHT_MCP_TOOL_PROFILE=essential` and `elementor-design` now expose
  `stonewright-elementor-v3-container-schema` when the WordPress MCP endpoint
  provides it.
- `stonewright-wordpress-mcp-status` now reports the post-filter proxied tool
  count, making startup diagnostics reflect the real visible tool surface.
