# Stonewright 1.0.0-alpha.19

Release date: 2026-06-16

## Highlights

- Adds `stonewright/design-implementation-contract` so agents can fetch one
  compact contract for global styles, section batching, native Elementor widget
  mapping, token-efficient batch tools, and hard visual failure conditions.
- Returns the same contract from `stonewright/context-bootstrap`,
  `stonewright/workflow-preflight`, and
  `stonewright/elementor-v3-capabilities-summary` so new MCP sessions start with
  the fast path.
- Tightens strict DesignSpec style fidelity checks so radius aliases cannot
  bypass provenance requirements.
