# Stonewright 1.0.0-alpha.28

Released: 2026-06-16

## Highlights

- Direct WordPress MCP connections now start in essential tools mode by default
  when the option has not been saved yet.
- New sessions see the compact Elementor, Gutenberg, content-model, WP-CLI, and
  workflow fast-path tools first, reducing startup payload and token use.
- Full tool discovery remains available by turning essential tools mode off in
  Stonewright configuration.

## Changed

- `AbilityRegistry::enabled_abilities()` now treats an unset
  `stonewright_essential_tools_mode` option as enabled.
- Admin settings, REST settings, `stonewright/workflow-preflight`, and
  `stonewright/tool-profile` now report the same default.
