# Stonewright 1.0.0-alpha.24

Release date: 2026-06-16

## Highlights

- Adds `stonewright-wordpress-mcp-status`, a local companion diagnostic tool
  that remains visible even when the WordPress MCP endpoint is unavailable.
- Makes companion startup resilient: if proxied WordPress MCP discovery fails,
  agents still get setup-profile plus direct `stonewright-wp-cli-*` tools.
- Adds the diagnostic tool to setup-profile visibility checks so new sessions
  can quickly tell whether missing WordPress tools are a connection issue.
