# Stonewright 1.0.0-alpha.38

Released: 2026-06-16

## Highlights

- Reduced MCP startup context by moving the server description to a compact
  fast-start guide that points agents to bootstrap, preflight, and profile tools
  instead of injecting the full visual discipline on initialize.
- Filtered server tool exposure through the current Stonewright public ability
  set, so essential mode and disabled abilities are respected before adapter
  registration work begins.

## Changed

- Server startup now emphasizes the fastest safe design path: global styles
  first, native Elementor widgets, section-by-section builds, schema checks, and
  batch mutation.
- MCP server registration now uses the same current public ability surface as
  Stonewright registration state.

## Fixed

- Custom site instructions are included once in the MCP server description
  instead of being appended twice.
