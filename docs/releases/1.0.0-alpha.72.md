# 1.0.0-alpha.72

## Highlights

- **Error self-repair loop:** failed abilities stamp `error_code` + `error_message` into audit `_meta`; recurring errors at task-start include concrete `repair` hints (`RemediationHints`); Audit Log UI shows the cause.
- **Direct error auditing:** failed Direct tools write to `audit-direct.jsonl`; task-start returns `recurring_errors`.
- **Pluginless Elementor:** `stonewright-elementor-status|data-get|data-update` over tokenized WP-CLI (JSON on stdin, mandatory file backup, best-effort CSS flush).
- **Gutenberg validate** + three built-in skills (copy-if-missing) + no-hallucination guidance.
- **AGENTS.md bootstrap:** managed `~/.stonewright/AGENTS.md` + read-only `stonewright-agents-md-sync`.
- **Direct write gate:** writes require a prior `stonewright-task-start` (opt-out `STONEWRIGHT_DIRECT_REQUIRE_TASK_START=off`).
- README Elementor-first with capability tables (308 plugin / 98 Direct).
