# Stonewright 1.0.0-alpha.48

Released: 2026-06-16

## Highlights

- Keeps compact profile diagnostics available even when the WordPress MCP
  endpoint cannot be reached.

## Changed

- `stonewright-wordpress-mcp-status` now reports the normalized `tool_profile`,
  `profile_expected_tool_count`, and `profile_missing_tool_names` before remote
  WordPress tool registration succeeds.
- Failed connections now keep selected-profile recovery hints instead of
  returning a null profile.

## Verification

- `npm test -- mcp-server`

## Assets

- `stonewright-1.0.0-alpha.48.zip`: installable WordPress plugin package.
- `stonewright-companion-1.0.0-alpha.48.tgz`: Node companion package.
