# Stonewright 1.0.0-alpha.27

Release date: 2026-06-16

## Highlights

- `stonewright/workflow-preflight` now skips the full Elementor capability
  summary for non-Elementor tasks.
- Content-model preflight responses for ACF, CPT UI, Pods, WooCommerce, and
  WP-CLI work stay smaller and faster while preserving batch-first guidance.
- Non-Elementor responses include `elementor.included=false` plus
  `request_tool=stonewright/elementor-v3-capabilities-summary` so agents can
  fetch Elementor context only when needed.
