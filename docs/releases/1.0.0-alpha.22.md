# Stonewright 1.0.0-alpha.22

Release date: 2026-06-16

## Highlights

- Adds companion-side proxied tool profiles through
  `STONEWRIGHT_MCP_TOOL_PROFILE`, so stdio MCP sessions can start compact
  before the client loads a huge WordPress tool list.
- Keeps direct `stonewright-wp-cli-*` companion tools local and deduplicated
  when the WordPress MCP endpoint also exposes matching ability names.
- Updates setup profile, WordPress admin snippets, and docs to recommend
  `STONEWRIGHT_MCP_TOOL_PROFILE=essential` for fast general WordPress,
  Elementor, Gutenberg, content-model, and WP-CLI work.
