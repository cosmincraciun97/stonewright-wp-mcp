# Stonewright 1.0.0-alpha.33

Released: 2026-06-16

## Highlights

- Added `stonewright/media-list` so agents can search existing WordPress media
  by title, filename slug, caption, description, alt text, and mime before
  uploading Figma/design assets again.
- Visual Elementor preflight now tells agents to search media first, then batch
  upload only missing assets.

## Changed

- Added the read-only media search ability to essential and Elementor design
  profiles.
- Kept compact startup under 60 tools by using `media-upload-batch` as the
  essential upload path for both single and multi-file uploads.
- Added contract and unit coverage for media search and essential-mode exposure.
