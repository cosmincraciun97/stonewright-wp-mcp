# Stonewright 1.0.0-alpha.42

Released: 2026-06-16

## Highlights

- Made compact profile diagnostics more precise so agents can tell the
  difference between profile-hidden WordPress tools and local companion tools
  that are already available.

## Fixed

- `profile_filtered_tool_count` now excludes companion-owned local tools such as
  `stonewright-wp-cli-status`.
- `stonewright-wordpress-mcp-status` now returns a compact
  `profile_filtered_tool_names` sample plus a recovery hint for switching
  `STONEWRIGHT_MCP_TOOL_PROFILE` only when a needed tool is profile-hidden.
