# Stonewright 1.0.0-alpha.7

Release date: 2026-06-11

This alpha tightens Stonewright's first-pass building guidance for Elementor
and Gutenberg while keeping the security envelope unchanged.

## Highlights

- Adds Elementor widget schema tab grouping for Content, Style, Advanced, and
  unknown controls.
- Adds shared Advanced tab guidance covering positioning, z-index, motion
  effects, transform, backgrounds, borders, masks, responsive controls,
  attributes, CSS IDs/classes, order, alignment, width, margin, and padding.
- Updates agent instructions and built-in skills so agents inspect widget
  schemas before writing settings and name only major parent containers.
- Adds native Gutenberg/block-theme workflow guidance for `theme.json`,
  templates, template parts, patterns, block supports, and Create Block Theme
  handoff.
- Strengthens the public README introduction with release badges and a compact
  Stonewright capability summary.
- Fixes ability truth matrix write/read detection so guidance strings do not
  mark read-only abilities as write operations.

## Release Assets

- `stonewright-1.0.0-alpha.7.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.7.tgz`: Node companion package for WP-CLI,
  health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
