# Stonewright 1.0.0-alpha.32

Released: 2026-06-16

## Highlights

- Workflow preflight is leaner for Figma/Elementor starts: it returns a compact
  design contract reference by default instead of the full contract blob.
- Agents can still request the full design implementation contract inline with
  `include_design_contract=true`, or call `stonewright/design-implementation-contract`
  when they need every rule.

## Changed

- Added `include_design_contract` to `stonewright/workflow-preflight`.
- Added compact `design_contract_ref` with global-style tools, section batch
  limits, MCP tool name, and load guidance.
- Updated workflow efficiency tests to lock the token-saving default behavior.
