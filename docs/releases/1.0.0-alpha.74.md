# 1.0.0-alpha.74

## Highlights

- **Reliable MCP surface:** Setup Apply-now verifies persistence, generated
  client snippets use the saved value, and task-start reports it explicitly.
- **Bootstrap by default:** fresh plugin, companion, and Direct/pluginless
  installs start compact. Task-start activates a task profile only for the
  current session and never overwrites the site preference.
- **Direct progressive discovery:** Bootstrap stays at eight tools and expands
  to compact Elementor/design, Gutenberg, content-model, site-admin, or general
  profiles instead of loading all 98 Direct tools.
- **Deterministic Elementor repair:** schema failures point back to exact typed
  schema requests; blocked raw writes are non-retryable and cannot trigger a
  repeated PHP fallback loop.

## Upgrade notes

- Update the plugin and companion together. Older companions do not understand
  session task profiles and require a restart after upgrading.
- Existing sites retain their selected Setup surface. Fresh installs default to
  Bootstrap; select it manually on an upgraded site if desired.
- After updating an MCP client, restart or reload its MCP servers, call
  `stonewright-task-start`, and honor `tools/list_changed` or the returned
  `re_list_instruction`.
