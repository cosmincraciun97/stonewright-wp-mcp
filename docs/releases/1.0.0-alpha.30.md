# Stonewright 1.0.0-alpha.30

Released: 2026-06-16

## Highlights

- Compact profiles now include `stonewright/security-create-one-time-link`,
  giving external browser MCPs a fast way to enter wp-admin for inspection and
  QA when the current user has permission.
- Essential mode stays under 60 tools by replacing the older bulk-create helper
  with the faster `stonewright/content-bulk-upsert-posts` path.

## Changed

- Added one-time admin access link visibility to the WordPress-side essential
  profile, Elementor design profile, site-admin profile, and companion proxied
  profiles.
- Kept `stonewright/content-bulk-create` available outside essential mode, but
  removed it from compact startup discovery.
