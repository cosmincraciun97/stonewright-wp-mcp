# Stonewright 1.0.0-alpha.23

Release date: 2026-06-16

## Highlights

- Changes generated companion setup snippets to use the versioned GitHub
  release tarball through `npx`, so fresh MCP sessions work even before npm
  package publishing is configured.
- Keeps `STONEWRIGHT_MCP_TOOL_PROFILE=essential` in the generated config, so
  startup remains compact and fast.
- Wires companion MCP server info, WordPress-MCP client info, and HTTP health
  responses to the current app version instead of the stale alpha.1 placeholder.
