# Stonewright 1.0.0-alpha.18

Release date: 2026-06-16

This alpha improves fast design-to-Elementor work. It adds stricter style
fidelity checks, native gallery lightbox settings, and lower-token WP-CLI batch
guidance for repeated content-model and plugin command work.

## Highlights

- Adds `style_policy=strict` support for DesignSpecs so borders, radius,
  shadows, and filters need measured provenance before render.
- Updates MCP-facing guidance so visual agents build one section batch at a
  time, set global-style plans first, and avoid invented card chrome.
- Renders Basic Gallery specs with native `gallery_link` and `open_lightbox`
  controls when media-file lightboxes are expected.
- Adds compact WP-CLI batch and summary response guidance for repeated CPT UI,
  ACF, post, meta, option, term, and plugin commands.

## Release Assets

- `stonewright-1.0.0-alpha.18.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.18.tgz`: Node companion package for
  WP-CLI, batch WP-CLI, health checks, optional MCP proxy, setup profiles, and
  Stonewright MCP prompts.
- `SHA256SUMS.txt`: checksums for release assets.
