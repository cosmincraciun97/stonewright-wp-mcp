# Stonewright 1.0.0-alpha.43

Released: 2026-06-16

## Highlights

- Keeps site skill playbooks reachable in compact MCP profiles so token-sensitive
  clients can load one matched playbook without full tool discovery.

## Changed

- `stonewright/skills-get` is now part of the WordPress essential ability set
  and the shared compact profile base.
- The essential profile stays under the compact startup cap by keeping
  `stonewright/media-list` plus `stonewright/media-upload-batch` and leaving
  `stonewright/media-get` for full/specialist discovery.
- Companion proxy profiles now keep `stonewright-skills-get` visible for
  `essential`, `elementor-design`, `content-model`, `gutenberg`, `wp-cli`, and
  `site-admin` profiles.
- `stonewright/workflow-preflight` now recommends `stonewright/skills-get` and
  adds a call-sequence step when matched site skills or plugin specializations
  are present.
- `stonewright-setup-profile` includes `stonewright-skills-get` in tool
  visibility checks for fresh MCP sessions.

## Verification

- `composer test -- --filter WorkflowEfficiencyAbilitiesTest`
- `npm test -- mcp-server setup-profile`

## Assets
