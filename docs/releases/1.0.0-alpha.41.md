# Stonewright 1.0.0-alpha.41

Released: 2026-06-16

## Highlights

- Improved companion startup diagnostics for token-sensitive MCP clients by
  showing which compact profile filtered the proxied WordPress tool surface.

## Fixed

- `stonewright-wordpress-mcp-status` now includes `tool_profile`,
  `remote_tool_count`, `proxied_tool_count`, and
  `profile_filtered_tool_count`.
- Agents can distinguish a broken WordPress MCP connection from a healthy
  compact profile that intentionally hides specialist tools.
