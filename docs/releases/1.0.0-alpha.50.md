# Stonewright 1.0.0-alpha.50

Released: 2026-06-17

## Highlights

- Keeps strict `low-tools` MCP sessions under the small-client startup budget while making direct WP-CLI background jobs visible.
- Improves long-running ACF, CPT UI, cache, import, plugin, media, and content work in Antigravity, Gemini API, and other strict tool-cap clients.

## Changed

- `STONEWRIGHT_MCP_TOOL_PROFILE=low-tools` now exposes `stonewright-wp-cli-job-start` and `stonewright-wp-cli-job-status`.
- Low-tools startup keeps duplicate legacy aliases and lower-priority discovery tools out of the initial surface to preserve the under-30 tool budget.
- The WordPress-side `stonewright/tool-profile` low-tools profile includes guarded WP-CLI job tools when essential mode is enabled.
