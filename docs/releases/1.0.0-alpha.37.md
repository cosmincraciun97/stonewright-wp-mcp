# Stonewright 1.0.0-alpha.37

Released: 2026-06-16

## Highlights

- Added a compact Elementor V3 container schema tool so agents can inspect compatible
  layout, style, Advanced, alias, and blocked-key guidance before section
  writes without loading broad widget schemas.
- Mapped common flex shorthands to Elementor's actual `flex_*` container keys
  during direct container writes, reducing layout retries.

## Added

- `stonewright/elementor-v3-container-schema`

## Changed

- Elementor design tool profiles and workflow preflight now include the
  container schema in the fast path.
- Essential tools mode remains under the compact startup limit by swapping out
  the less common `elementor-v3-save-template` slot.
