# Stonewright 1.0.0-alpha.10

Release date: 2026-06-11

This alpha tightens Stonewright's visual implementation workflow so
design-derived WordPress pages have a clearer evidence gate before signoff.

## Highlights

- Adds `visual_build_gate` to `stonewright/context-bootstrap` and
  `stonewright/workflow-preflight`.
- Requires pixel-matching tasks to collect a reference token table, existing
  media reuse audit, and section implementation plan before the first write.
- Requires desktop, tablet, and mobile screenshot deltas plus logged-out public
  viewport checks before visual completion.
- Strengthens Elementor guidance for section-sized first passes, existing asset
  reuse, `.e-con-inner` layout inspection, and screenshot-driven fixes.
- Keeps the safety envelope unchanged: writes still require permissions,
  context tokens, backups where required, validation, and production-safe
  confirmation tokens.

## Release Assets

- `stonewright-1.0.0-alpha.10.zip`: installable WordPress plugin package with
  production Composer dependencies.
- `stonewright-companion-1.0.0-alpha.10.tgz`: Node companion package for
  WP-CLI, health checks, and optional MCP proxy.
- `SHA256SUMS.txt`: checksums for release assets.
