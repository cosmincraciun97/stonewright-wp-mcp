## Purpose
Elementor uses Google Fonts in the core to allow you to use fonts from their library. While many of the fonts in their repository are supported, fonts that have been recently added may not yet be in the Elementor typography settings. The list of currently supported Google fonts is located here. This list is frequently updated. Please check the list if unable to locate a font you are wishing to use.
