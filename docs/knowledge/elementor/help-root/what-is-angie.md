## Purpose
Angie is a context aware AI Agentic plugin designed to guide users through the entire process of creating and maintaining a WordPress website.

## Use this when
- You can then use these elements anywhere in your site
- In addition to these vibe coding abilities, Angie can also help you create and maintain your site
- It can create posts, images and pages, as well as help you keep your plugins up-to-date
- Create an online store

## Settings highlights
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
