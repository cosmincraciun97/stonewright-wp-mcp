## Purpose
If you can’t edit in Elementor for any of the following reasons, follow this guide to resolve the most common issues.

## Use this when
- If the URL in the back-end is different from the one in the front-end it can cause the blank page error to occur
- For details, refer to Giving WordPress Its Own Directory – Advanced Administration Handbook
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.

## Settings highlights
- Before you begin – Check if you meet system requirements – See System Requirements To Use Elementor. Ensure version compatibility – If you are using both the free version of Elementor and Elementor Pro, make sure the versions are compatible. Using newer version with the outdated version can cause conflicts.
- Note – Essential, Advanced, Advanced Solo, and Elementor App subscriptions include access to the Elementor chatbot. Expert, Agency, Elementor One, Elementor Hosting, and Priority Support plans also include access to the Elementor support team.
- Advanced – Console or Server Issues
- Check the back – end and front-end urls
- If the URL in the back – end is different from the one in the front-end it can cause the blank page error to occur. For details, refer to Giving WordPress Its Own Directory – Advanced Administration Handbook.
- Cross – origin issues
- Change the X – Frame-Options from DENY to SAMEORIGIN. Ask your hosting company to solve this.

## Limits / gotchas
- Essential, Advanced, Advanced Solo, and Elementor App subscriptions include access to the Elementor chatbot. Expert, Agency, Elementor One, Elementor Hosting, and Priority Support plans also include access to the Elementor support team.
- You may be experiencing a memory issue. Check WP Memory Limit and increase it if needed. For details, refer wp-config.php – Common APIs Handbook.
- If your theme uses a preloader, you may need to disable the preloader within the theme’s options or within Customizer. For details, see Is Elementor Compatible With All WordPress Themes?.
