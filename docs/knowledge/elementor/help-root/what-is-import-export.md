## Purpose
Whatever the reason, Elementor provides Import and Export Kit features to help you.

## Use this when
- Use the checkboxes to select what plugins to import and click Next
- Use the checkboxes to select what parts of the site your want to import and click Import
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.

## Settings highlights
- Content – Export pages, posts, custom post types, menus, taxonomies, and media format. Note that if you choose Save media to cloud (Advanced plan and up) you can’t save the file as a zip; only to the cloud Library. You can include everything or select individual pages or content groups. (Granular options are Expert plan and above).
- Templates – Save site parts such as headers, footers, Theme Builder templates, and Global Widgets.
- Settings & Configurations – Global fonts, global colors, theme style, layout, lightbox, background settings, and active experiments.Export custom files such as custom fonts, icons, and custom code (Pro and above).Export the Theme: You can export public WordPress themes. (Pro and above).
- Plugins – Select which plugins to export. Note that all plugins are required for website templates to export successfully.
- Export as zip – Download your website template as a zip file. You can then upload this zip file to your new site. Note: If you selected Save media to cloud under Content, you won’t be able to export as a zip.
- Save to library – Save the website template to your virtual library. You can then import the site from the cloud.
- Open the Library – This will allow you to import a website template from your cloud library
- Import a zip file – Upload a zip file of a website template.

## Limits / gotchas
- Plugins: Select which plugins to export. Note that all plugins are required for website templates to export successfully.
- Export as zip: Download your website template as a zip file. You can then upload this zip file to your new site. Note: If you selected Save media to cloud under Content, you won’t be able to export as a zip.
- Drag and drop the exported zip file into the window or click Select File and navigate to the zip file.You may receive a warning about importing zip files. If you created the zip file or trust the source of the zip file, continue with the import.
- Click Import.You may receive a warning that the kit contains svg files.If you created the export kit or trust the source of the export kit, Enable svg imports.
