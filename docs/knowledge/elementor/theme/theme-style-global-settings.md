## Purpose
In order to apply the Theme Style settings to most of Elementor’s elements and widgets as well, you will need to disable Elementor’s default colors and fonts.

## Use this when
- If you prefer, you can force these Theme Style settings to also apply to Elementor-built elements as well
- Use the checkboxes to Disable Default Fonts and Disable Default Colors
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.

## Settings highlights
- Note – Elementor Pro users can also set Global Custom CSS which lets you apply custom CSS rules to your entire site.
- Text Color – Choose the color of the default text
- Typography – Change the typography options for the default text
- Paragraph Spacing – Set the amount of default spacing after a paragraph (margin-bottom)
- Color – Choose the default color of links for both normal and hover states
- Typography – Change the default typography of links for both normal and hover states
- Color – Choose the default color for each Heading (H1, H2, H3, H4, H5, and H6)
- Typography – Set the default typography options for each Heading (H1, H2, H3, H4, H5, and H6)

## Limits / gotchas
- Elementor Pro users can also set Global Custom CSS which lets you apply custom CSS rules to your entire site.
- Some elements won’t get affected because Elementor doesn’t allow editing them (e.g fieldset, input[type=”reset”] and more)
