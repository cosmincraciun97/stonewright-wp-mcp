## Purpose
Templates are a great way to make website building more efficient by allowing you to reuse site parts and pages that you’ve created.

## Use this when
- This means that Elementor users can use these templates for any site they’re working on
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.
- Customizing specific styles, responsiveness, and display logic for elements across devices.

## Settings highlights
- About privacy – Cloud Templates and Cloud Website Templates are tied to the individual user’s Elementor account and Elementor Pro subscription. This means that only the user who connects their account to the site can access or upload templates to their personal Cloud space. Other WordPress users on the same site will not have access to this service unless they log in with their own Elementor account, and have an active Elementor Pro subscription.
- Site templates – These templates are available to all users with an Elementor account. They are stored locally, on your site and can be reused for that site’s pages.
- Cloud templates – These templates are available for Elementor users with an Elementor Pro subscription. This means that Elementor users can use these templates for any site they’re working on.
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- It’s also important to note that using Cloud Templates is separate from activating Elementor Pro or any other subscription. Each user must have their own Elementor account and subscription to access these services.
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
