## Purpose
Elementor websites use a number of templates, some of which represent website parts that are used on multiple webpages. For example, headers and footers. When you create site parts, you have the option of displaying them on all your pages or selecting certain pages for them to appear on.

## Use this when
- Templates can appear on all or some of your pages
- For instance, you can have the same header and footer appear on all your site’s pages or you can create a special header for your product pages
- Create a site part template
- If you want to limit the template to certain pages, use the left dropdown to determine if you’re going to Include or Exclude pages

## Settings highlights
- Create a site part template. See – What is the Theme Builder? for more information.
- Entire Site – Use to select every page on the site.
- Singular – Use to select individual pages or groups of pages on the site. After selecting Singular, you need to narrow your selection:
- All Singular – Any page that doesn’t fall under another category such as Archives.
- Front Page – The Home page.
- Posts – All Posts by default with the option to manually select specific posts.
- In Category – All posts in a selected category.
- In Child Categories – all posts in selected subcategories.

## Limits / gotchas
- If the condition conflicts with the conditions of another site part, a warning appears. For example, in the example below, we’re trying to include a header on all pages while there is already another header set to appear on the site’s pages.
- If you want to limit the template to certain pages, use the left dropdown to determine if you’re going to Include or Exclude pages.
