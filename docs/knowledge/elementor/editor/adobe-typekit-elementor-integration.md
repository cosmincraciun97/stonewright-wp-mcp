## Purpose
Integrates your Adobe Fonts (formerly TypeKit) account with Elementor to provide instant access to your Adobe TypeKit project fonts inside the Elementor Editor's typography settings.

## Use this when
- You want to use premium Adobe/TypeKit fonts in your Elementor page designs.
- Syncing project-specific fonts from an Adobe Creative Cloud account to WordPress.

## Settings highlights
- Elementor > Settings > Integrations (navigation path in WP Admin)
- Adobe Fonts (TypeKit) setting panel
- TypeKit Project ID text input field
- Get Project ID validation button

## Limits / gotchas
- Adobe's CDN propagation delay: It may take several minutes for Adobe to update their network and sync newly added project fonts into Elementor.
- Requires an active Adobe Fonts subscription and a valid Project ID.
