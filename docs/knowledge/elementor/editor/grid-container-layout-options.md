## Purpose
The Grid Container allows you to quickly align the elements on your page but contains enough options to keep your designs fresh and interesting. Click on the hotspots below to explore the Grid Container’s layout options.

## Use this when
- Minimum Height – Use the slide to adjust the minimum height of the container
- Learn more about measurement units
- Grid Outline – By default, set to Show which will allow you to see the outline of the cells in a grid
- Hide will make the cell lines invisible

## Settings highlights
- Grid – A Grid Container is created with an equal number of symmetric rows and columns.
- Flexbox – A Flexbox Container contains rows and columns in a pattern chosen by the user.
- Container Width – A dropdown menu with two choices:
- Measurement dropdown menu () – Use the dropdown to toggle between measuring rows in fractions (FR) and in custom measurements. Custom measurement allows you to set row height by standard units of measurement (e.g. pixels(PX), EM, REM etc.) or as CSS properties. Custom measurement is also used to create different size rows. This is done by selecting custom measurement and entering a value for each row. For example, for three rows you could enter the values “200PX 1FR 400PX”. This would create three rows of these respective heights.

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
