## Purpose
Configure your Elementor editor interface to display in a different language than your website's frontend. This allows builders to work in their preferred language while maintaining the site's original language settings for visitors.

## Use this when
- Your website is in one language but you prefer editing in another
- Working on multilingual sites with different language preferences per team member
- You need the editor UI in a specific language for comfortable workflow
- Managing content across multiple language versions of a site
- Collaborating with international teams who speak different languages

## Settings highlights
- Language selection available in editor preferences or account settings
- Independent from WordPress admin language and site language settings
- Applies only to the Elementor editor interface, not front-end content
- Supports multiple languages available in Elementor's localization options
- Changes take effect immediately upon selection and editor reload
- Language preference syncs across your Elementor account

## Limits / gotchas
- Editor language setting does not affect website content language or visitor experience
- Some third-party plugins may not respect the editor language preference
- Requires the selected language to be installed/available in your Elementor installation
