## Purpose
Using variables can help speed up your workflow and increase efficiency. The Variable Manager gives you greater control over your variables.

## Use this when
- When you create a variable it is added to the Variable Manager where you can edit or delete it
- Create a variable in the Variable Manager
- Click Create a variable if this is your first variable or the new variable icon – +
