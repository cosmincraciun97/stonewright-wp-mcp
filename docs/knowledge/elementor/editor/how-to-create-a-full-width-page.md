## Purpose
A full-width page layout extends content across the entire screen width (100%), removing standard sidebar constraints. This design approach maximizes visual real estate and creates immersive experiences, commonly used for landing pages, hero sections, and showcase content that benefits from expanded horizontal space.

## Use this when
- Creating landing pages requiring maximum visual impact
- Designing hero sections with full-bleed background images
- Building portfolio or gallery showcases needing expansive layouts
- Developing call-to-action sections with prominent content
- Implementing canvas-style pages without theme containers

## Settings highlights
- Layout tab configuration for full-width settings
- Canvas page template option for distraction-free editing
- Container and section width controls
- Padding and margin adjustments for edge spacing
- Responsive breakpoint settings for mobile optimization
- Background image/video full-width extension options
- Z-index management for layered full-width elements

## Limits / gotchas
- Full-width layouts may impact readability on ultra-wide displays without proper content constraint
- Mobile responsiveness requires careful testing as full-width content can become cramped on small screens
- Some theme frameworks may have conflicting wrapper styles that override Elementor's full-width settings
