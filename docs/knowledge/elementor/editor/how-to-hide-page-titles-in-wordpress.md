## Purpose
If you’re using Elementor, you can hide titles with one click, no need to use any code or any other plugin.

## Use this when
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.
- Customizing specific styles, responsiveness, and display logic for elements across devices.

## Settings highlights
- <h1 class="entry – title">Title</h1>
- If the class name is “abc – page-title” for example, then you would type h1.abc-page-title into the Page Title Selector in the Elementor > Settings > Style area mentioned above.
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- Also note that when working in Elementor Full Canvas mode, the page name is never displayed.
- that if your theme is using a different class for the title, this option won’t automatically work and you will have to manually set it.
- Make a note of the H1 class name, which is shown in quotes
