## Purpose
Now that you know how to create a new page, we’ll show you how to build your first page in Elementor using flexbox containers. Containers replace the previous system of sections and columns, providing an easier and more flexible way to build a web page.

## Use this when
- Now that you know how to create a new page, we’ll show you how to build your first page in Elementor using flexbox containers
- Containers replace the previous system of sections and columns, providing an easier and more flexible way to build a web page
- First thing we’ll need to do is create a container to hold your hero section
- Click the icon to create a layout for your container

## Settings highlights
- Position of the background image – Center-Center
- Size – Cover Now one big image will cover the entire section.
- Family – Montserrat
- Line Height – 26PX(Learn more about typography)

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
