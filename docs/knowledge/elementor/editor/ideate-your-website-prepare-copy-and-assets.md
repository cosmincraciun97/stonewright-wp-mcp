## Purpose
This article guides users through the planning phase before building a website, focusing on gathering and organizing content materials needed for design implementation in Elementor's editor. It emphasizes the importance of preparing assets and copy before entering the builder.

## Use this when
- Starting a new website project and need to organize assets beforehand
- Planning page structure and determining what copy/images you'll need
- Preparing materials before opening the Elementor editor
- Coordinating team content collection efforts
- Establishing a content inventory for multiple pages

## Settings highlights
- Asset organization strategies for images, videos, and graphics
- Copy preparation guidelines for different page sections
- Brand guidelines documentation methods
- Content hierarchy planning for page layouts
- File naming conventions for easy asset management
- Color and font documentation processes
- Stakeholder feedback collection approaches

## Limits / gotchas
- This is preparatory work — it does not replace actual page building in Elementor
- Incomplete asset gathering can slow down the design process significantly
- Poor organization of materials may cause confusion during implementation phase
