## Purpose
Elementor's element addition feature allows users to populate pages with widgets and content blocks through an intuitive drag-and-drop interface. This represents the fundamental action for building layouts within the visual editor, enabling designers to incorporate text, images, forms, and specialized widgets into their web designs.

## Use this when
- Building page layouts from scratch or adding content to existing designs
- Inserting widgets like buttons, headings, images, or forms
- Creating complex sections with multiple nested elements
- Organizing content through containers and flexbox structures
- Designing responsive pages with columns and sections

## Settings highlights
- Drag-and-drop element placement from the elements panel
- Widget library containing dozens of pre-built components
- Inline editing capabilities for quick content modifications
- Advanced tab options for custom styling and spacing
- Display conditions to show/hide elements based on user criteria
- Nesting support for creating hierarchical layouts
- Responsive design controls for mobile and tablet views
- Element duplication and cloning shortcuts
- Z-index management for layering elements

## Limits / gotchas
- Performance may degrade with excessive nested elements on a single page
- Some widgets require Elementor Pro subscription for full functionality
- Element compatibility varies depending on active plugins and theme
- Mobile responsiveness requires manual adjustment across different breakpoints
