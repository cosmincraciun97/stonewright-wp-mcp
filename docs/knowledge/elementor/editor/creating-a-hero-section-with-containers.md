## Purpose
A Hero section is a prominent full-width container at the top of a webpage designed to make an immediate visual impact. It typically combines background imagery, text overlays, and call-to-action buttons to introduce your brand or message to visitors upon page load.

## Use this when
- Creating landing pages that need strong initial visual appeal
- Building homepages requiring immediate brand messaging
- Designing product showcase pages with featured content
- Setting up service introductions with compelling visuals
- Establishing memorable first impressions for marketing campaigns

## Settings highlights
- Background image or video customization with positioning controls
- Text overlay styling with font, color, and typography options
- Container layout configuration using flexbox or grid systems
- Call-to-action button placement and styling
- Height and spacing adjustments via padding and margin settings
- Responsive design controls for mobile and tablet viewports
- Advanced effects including parallax, transparency, and blur options
- Z-index management for layered element positioning
- Display conditions to show/hide based on visitor criteria

## Limits / gotchas
- Large background images may impact page load performance; optimize file sizes accordingly
- Text readability requires sufficient contrast against background imagery; test across devices
- Hero sections consume significant viewport space on mobile; ensure responsive layouts don't overwhelm smaller screens
