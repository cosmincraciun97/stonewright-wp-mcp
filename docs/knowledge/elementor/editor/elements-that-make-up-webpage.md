## Purpose
Creating your first website can feel like a daunting task. You’ll need to design several site parts and sections to build a professional and stunning website.

## Use this when
- The header part is located at the top of the web page and its content rarely changes
- The header section typically contains the business title, logo, login information, and a menu, often with navigation drop-down links
- You can use these elements to navigate to other parts of the website
- Alternatively, discover how you can use a website kit with ready-to-use headers, footers and professionally designed pages

## Settings highlights
- The hero section often contains a call – to-action for the visitor!
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
