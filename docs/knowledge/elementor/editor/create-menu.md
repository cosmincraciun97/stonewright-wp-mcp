## Purpose
A website’s navigation menu allows users to quickly jump between different sections of your website. Menu items generally link to the most important pages such as, Home, Contact Us, Blog etc.

## Use this when
- Create a top level menu
- Use the checkmarks to select which pages will be included in the menu
- Use the Add menu items menu to choose between adding Pages, Posts, Custom Links and Categories to the menu
- Use the checkmarks to select the subpages, and click Add to Menu

## Settings highlights
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
