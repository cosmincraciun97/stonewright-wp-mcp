## Purpose
After you’ve set up your site, one of the first things you’ll want to do is build your first page. Even if you chose to start with a kit, this is a great way to understand the editor, which will help you customize your kit. Before starting, you’ll need to create a new page.

## Use this when
- After you’ve set up your site, one of the first things you’ll want to do is build your first page
- Even if you chose to start with a kit, this is a great way to understand the editor, which will help you customize your kit
- Before starting, you’ll need to create a new page
- Click the icon to create a section

## Settings highlights
- Sections – the largest building blocks, which house the content inside columns.
- Columns – sit inside of Sections and house Widgets.
- Widgets – are placed inside Columns and are functional site components.
- Right – click on this image and save it for the hero section.

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
