## Purpose
Many Elementor Widgets offer the ability to edit text color and typography settings. These design options are generally located in the widgets’ Style tab.

## Use this when
- Use the Family dropdown menu to choose what type of font you want to use
- If you know the name of the font family, you can enter it in the text box at the top of the dropdown menu
- Use the Size slider to set how big you want the font
- To learn more about the different measurement systems, check out Units of measurement

## Settings highlights
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
