## Purpose
This article explains the mechanics of repositioning sections within the Elementor page builder interface. Users can rearrange the vertical order of page sections to restructure layout flow and content hierarchy without recreating elements from scratch.

## Use this when
- You need to change the visual order of major page content blocks
- Reorganizing page structure after initial design
- Moving sections between different parts of a page layout
- Restructuring without manually rebuilding elements
- Managing complex multi-section page designs

## Settings highlights
- Drag-and-drop section reordering in the editor canvas
- Navigator/Structure window for viewing page hierarchy
- Visual section selection and movement controls
- Undo/redo functionality for layout changes
- Responsive preview during repositioning
- Keyboard shortcuts for efficient workflow
- Copy and paste sections between pages
- Section nesting options with containers

## Limits / gotchas
- Section dependencies (conditional styling or dynamic content) may require manual adjustment after moving
- Complex interactions or animations tied to section position may need reconfiguration
- Nested containers require understanding parent-child relationships before moving
