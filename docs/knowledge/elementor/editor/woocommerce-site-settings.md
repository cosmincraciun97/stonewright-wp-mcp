## Purpose
Allows site administrators to assign default WooCommerce page endpoints directly from the Elementor Site Settings panel.

## Use this when
- Linking your custom Elementor cart, checkout, or account templates to active WooCommerce systems.

## Settings highlights
- Site Settings > WooCommerce (editor panel settings)
- Cart Page assignment dropdown
- Checkout Page assignment dropdown
- My Account Page assignment dropdown
- Purchase Summary Page dropdown

## Limits / gotchas
- Endpoints must correspond to valid, published WordPress pages containing the respective Elementor layout template blocks.
