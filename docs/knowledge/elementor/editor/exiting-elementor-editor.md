## Purpose
Exiting the Elementor Editor allows users to leave the visual building interface and return to their WordPress dashboard or website frontend. This fundamental navigation task helps designers and developers manage their workflow between editing and publishing states.

## Use this when
- You need to save changes and close the editor interface
- Returning to WordPress admin dashboard
- Switching between pages or posts
- Ending an editing session
- Accessing website settings outside the builder

## Settings highlights
- Close button locations in editor interface
- Save options before exiting
- Draft vs. published state management
- Return to dashboard shortcuts
- Preview before exit functionality
- Revision history access
- Page-specific exit workflows

## Limits / gotchas
- Unsaved changes may be lost if exiting without saving
- Different exit methods exist depending on editor version (V3 vs V4)
- Some workflows require specific exit procedures to preserve settings
