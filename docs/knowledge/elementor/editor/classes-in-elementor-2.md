## Purpose
The Elementor Editor is designed to help web creators work quickly and efficiently. As part of that mission, the Editor adopts the principles of Atomic Design. This system is based on the idea of creating small, reusable elements. In Elementor, these elements are called classes. This is in line with the vocabulary used by CSS.

## Use this when
- Select the class name under Create a new class
- Create a global class from a local class
- Create an element and add several properties to it
- For this example, we’ll create a heading and add alignment, font type and color

## Settings highlights
- Tip – It’s best to build your classes so there are as few conflicts as possible. These conflicts can not only waste your time, they can also slow down your site.
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- In the dropdown menu, select Create new “Main button”.Main_button appears in the Classes text box. Note that the class name Main_button is in green. That means you’re editing that class.
- In the CSS Classes text box, delete Main button.Note that the button returns to its default settings.
- In the dropdown menu, select Create “Secondary button”.Note the class name is in green meaning you’re editing the class.
