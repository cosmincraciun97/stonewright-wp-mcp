## Purpose
Allows users to switch the Elementor Editor UI interface from the default light theme to a dark theme style to reduce eye strain in low-light environments.

## Use this when
- You prefer a darker workspace theme for editing Elementor pages.
- Automating the dark theme activation according to system-level OS preferences.

## Settings highlights
- User Preferences (Editor Menu)
- Dark Mode option dropdown
- Auto option (inherits system OS theme settings)
- Dark option (forces dark theme)
- Light option (forces light theme)

## Limits / gotchas
- Editor dark mode only changes the Elementor workspace panels and settings UI; it does not change the actual front-end layout style of the website canvas.
