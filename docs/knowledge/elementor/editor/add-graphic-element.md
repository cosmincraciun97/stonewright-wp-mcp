## Purpose
This article explains how to incorporate visual elements — images and icons — into Elementor pages. It covers the tools and widgets available for displaying graphics, managing their properties, and styling them to enhance website visual design and user experience.

## Use this when
- You need to add photographs, illustrations, or graphic files to your page
- You want to insert icon symbols from Elementor's icon library
- You're working with image galleries or carousels
- You need to style images with effects, borders, or animations
- You're building layouts that rely on visual content as primary design elements

## Settings highlights
- **Image widget**: Upload and display single images with sizing options
- **Icon widget**: Select from extensive icon libraries with color and size controls
- **Image Box widget**: Combine images with text overlays and links
- **Gallery widget**: Create organized image collections with lightbox functionality
- **Image Carousel widget**: Display multiple images in rotating carousel format
- **Icon Box widget**: Pair icons with text for feature highlighting
- **Icon library access**: Browse and customize icon sets
- **Image properties**: Adjust dimensions, alignment, spacing, and responsive behavior
- **Effects and filters**: Apply blend modes, shadows, and transformations

## Limits / gotchas
- SVG support requires explicit enabling in site settings
- Icon customization is limited by selected icon font library
- Image file size impacts page load performance and bandwidth usage
- Responsive image sizing needs manual configuration for optimal mobile display
