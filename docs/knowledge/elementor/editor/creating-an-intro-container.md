## Purpose
One of the most fundamental sections on a website is an intro/splash section. This is generally a welcoming title with a tagline. It is also a good place to start interacting with the user by adding buttons to direct them to additional information or a purchase. In this tutorial, we will teach you how to create this using containers.

## Use this when
- To get started, first create a new page on your website and begin editing with Elementor
- In this step you will finally use the power of the flex properties in the container
- The widgets are all in place but now need to be positioned within the container properly
- Begin by editing your container

## Settings highlights
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
