## Purpose
Create curved text paths that follow custom shapes in Elementor. This feature allows designers to arrange text along SVG-based paths, enabling creative typography effects like circular text, wavy text, or text following any custom shape design.

## Use this when
- You need text to follow a curved or custom path for creative layouts
- Designing decorative headers or special typographic effects
- Creating circular badges or logos with curved text
- Building artistic designs that require text along specific shapes
- You want typography that adapts to non-linear paths

## Settings highlights
- SVG path input for defining custom text routes
- Text alignment options along the path (start, middle, end)
- Letter spacing and kerning controls
- Font size and color customization
- Path direction and offset adjustments
- Real-time preview of text placement on curve
- Support for custom SVG shape integration

## Limits / gotchas
- Requires understanding of SVG path syntax or access to SVG creation tools
- Complex paths may cause text readability issues on mobile devices
- Very long text strings may not fit properly on tight curves
- Accessibility considerations needed for screen readers interpreting curved text
