## Purpose
The Flip Box Widget helps you create animated boxes that flip to the other side, when a visitor hovers over them.

## Use this when
- The Flip Box Widget helps you create animated boxes that flip to the other side, when a visitor hovers over them
- For more details, see, Create a Background
- This helps search engines find and understand the price list, boosting SEO
- The title can also be tagged as a paragraph, span or div

## Settings highlights
- Graphic Element – Choose between None, Image or Icon to display a graphical Element in the front of the flip box. For more details, see Adding images and icons.
- Title & Description – Choose the title and description that appears in the front of the flip box
- Background Type – Choose Color, Image or Gradient as the background of the front of the flip box. For more details, see, Create a Background.
- Position – Select the position of the image, such as Top Center, Top Right, Center Center, etc.
- Attachment – Select from Default, Scroll, or Fixed
- Repeat – Choose from Default, No-repeat, Repeat, Repeat-x, or Repeat-y
- Size – Select from Default, Auto, Cover, or Contain
- Background Overlay – Choose a color for the overlay

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
