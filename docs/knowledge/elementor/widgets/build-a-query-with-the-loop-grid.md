## Purpose
Build dynamic loops of posts or custom content with custom query parameters. The Loop Grid widget displays repeated content based on filtering, sorting, and pagination rules you configure, enabling flexible archive and category pages without manual repetition.

## Use this when
- Creating blog archive or category pages with dynamic filtering
- Building product listings with custom sorting options
- Displaying related posts or filtered content collections
- Designing pages that need pagination and item limits
- Setting up taxonomy-based or search result pages

## Settings highlights
- Query builder to select post type, taxonomy, and custom parameters
- Filtering by categories, tags, authors, and custom fields
- Sorting options (date, title, relevance, custom fields)
- Pagination controls and items-per-page limits
- Layout customization for grid display
- Alternate template support for varied item layouts
- Dynamic tag integration for content population
- Search and taxonomy filter widget compatibility

## Limits / gotchas
- Query complexity may impact page load times on large databases
- Pagination resets filter selections unless configured with URL parameters
- Alternate templates require separate template design for each variation
- Custom field sorting requires ACF or similar plugin integration
