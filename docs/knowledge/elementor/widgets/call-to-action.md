## Purpose
The Call to Action (CTA) widget enables designers to create compelling conversion-focused sections that combine text, imagery, and buttons to prompt specific user actions. This widget is essential for marketing initiatives, promotional campaigns, and guiding visitors toward key conversion points on your website.

## Use this when
- Building landing pages that require strong conversion elements
- Creating promotional sections within larger page designs
- Designing email signup or download incentive areas
- Establishing clear next-step guidance for website visitors
- Combining visual elements with actionable button directives

## Settings highlights
- Content area supports text, headings, and descriptive copy
- Button configuration — customizable button text, styling, and link destinations
- Background options — image, color, or gradient background support
- Layout controls — alignment and spacing adjustments for content positioning
- Responsive design — breakpoint-specific adjustments for mobile and tablet views
- Typography styling — font family, size, color, and weight customization
- Advanced options — CSS classes, custom attributes, and hover effects
- Display conditions — show/hide rules based on user or page criteria

## Limits / gotchas
- Widget effectiveness depends heavily on clear, action-oriented copy and visual hierarchy
- Background images may impact load times; optimize file sizes accordingly
- Button contrast and readability are critical for accessibility compliance
