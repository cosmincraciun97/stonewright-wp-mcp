## Purpose
A loop grid displays multiple items from your WordPress database in a grid layout with customizable templates. This widget automatically queries and repeats content for posts, products, or custom post types across responsive columns, making it ideal for showcasing blog posts, portfolios, or product catalogs.

## Use this when
- You need to display multiple posts or products in a grid format
- You want dynamic content that pulls from your database automatically
- You're building archive pages, category pages, or portfolio sections
- You need alternate templates for different item types within the same grid
- You want pagination or filtering capabilities for large content collections

## Settings highlights
- Query builder to customize which items appear (posts, products, taxonomies)
- Grid layout controls for columns and responsive breakdowns
- Template selection for how each item displays
- Pagination settings to limit items per page
- Alternate template options for varied item styling within one grid
- Custom ordering and filtering by taxonomy, date, or metadata

## Limits / gotchas
- Performance may degrade with very large datasets; consider pagination limits
- Alternate templates require separate template creation beforehand
- Styling must account for responsive breakdowns on mobile devices
- Off-Canvas widgets in loop grids have specific integration requirements
