## Purpose
Provides a comprehensive index and functional breakdown of all custom widgets available in Elementor Pro for building dynamic Single Product layouts.

## Use this when
- Determining which specialized widgets to use for custom product catalog interfaces.

## Settings highlights
- Product Title widget control
- Product Images / Gallery widget control
- Product Price widget control
- Add to Cart / Custom Add to Cart widget control
- Product Rating widget control
- Product Stock / SKU Info widget control
- Product Content / Short Description widget control
- Product Meta widget control
- Dynamic Tags links

## Limits / gotchas
- Requires Elementor Pro.
- Can only be added inside Single Product templates or in specialized product archives.
