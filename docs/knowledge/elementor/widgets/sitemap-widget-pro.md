## Purpose
In the Source field, specify the source of the content to be included in the sitemap. The available sources will vary based on your selection in the Type field.If you choose Post Type, select Posts, Pages, or any available custom post types.If you choose Taxonomy, select Categories, Tags, Format, or any available custom taxonomies. ## Settings highlights
- David manages an e – commerce website that sells ceramics products. To improve the user experience and help customers easily find what they’re looking for, David decided to showcase a sitemap on the website. By creating a visual map of products and product categories, David provides a convenient way for visitors to navigate the store’s inventory. This enhances the website’s usability and as well as allows search engine optimization by ensuring all product pages are easily accessible and indexed by search engines.
- Add a Sitemap widget – Step-by-step
- Choose how to order the links – For Post Type selection, choose from Date, Title, Menu Order, or Random. For Taxonomy selection, choose from ID or Name.
- Set the Title tag to H1 – H6, Div, Span or Paragraph.
- Color – Choose color of the title textTypography: Customize title’s typography options, such as font, size, style, and more. For more details, see Typography.
- Color – Select color of the page links.Typography: Customize typography options such as font, size, style, and more for the page links For more details, see Typography.
- Color – Choose color of the list bullets.Style: Select style of the list bullet, including Disc, Circle, Square, or None.

## Limits / gotchas

- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
