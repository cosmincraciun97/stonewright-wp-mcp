## Purpose
Configures the standard Products widget to showcase custom selections of products on any site page.

## Use this when
- Featuring specific products, sales, or collections on a homepage or landing page.

## Settings highlights
- Columns count setting (1 to 12)
- Query source (Latest, Sale, Featured, Manual, Related)
- Order By setting (Date, Title, Price, Popularity, Rating, Random)
- Order direction setting (ASC, DESC)
- Pagination visibility toggle
- Column Gap / Row Gap style sliders (in pixels)
- Image, Title, Price, Rating, Button style states (Normal, Hover)

## Limits / gotchas
- Selecting 'Related' source requires placement inside a single product template.
- Requires active WooCommerce plugin.
