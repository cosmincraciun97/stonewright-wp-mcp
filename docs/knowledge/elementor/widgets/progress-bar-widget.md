## Purpose
The Progress Bar Widget allows you to add fully styled, animated progress bars to your page.

## Use this when
- This helps search engines find and understand the progress bar, boosting SEO
- The title can also be tagged as a paragraph, span or div
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.

## Settings highlights
- Title – Enter the title text that is displayed above the progress bar
- Title HTML Tag – Use the dropdown menu to designate the title of the progress bar as a header (H1-6).This helps search engines find and understand the progress bar, boosting SEO. The title can also be tagged as a paragraph, span or div
- Percentage – Set the completion percentage number
- Display Percentage – Use the switch to show or hide the actual percentage number text at the end of the bar
- Inner Text – Enter the text to be displayed within the bar
- Color – Change the color of the bar
- Background – Choose the color of the background of the bar
- Inner Text Color – Choose the color of the text to be displayed within the bar

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
