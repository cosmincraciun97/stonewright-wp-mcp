## Purpose
Utilizes dynamic tags inside the Elementor Editor to automatically fetch and display product-specific data from WooCommerce databases on product page templates.

## Use this when
- Designing a custom single product or shop archive layout template.
- Dynamically retrieving price, stock, ratings, cart contents, or checkout details into custom widgets.

## Settings highlights
- Product Price tag (fetches standard, original, or sale prices)
- Product Title tag
- Product Image tag
- Product Stock / SKU tag
- WooCommerce Cart page endpoint
- WooCommerce Checkout page endpoint
- WooCommerce My Account page endpoint

## Limits / gotchas
- Requires both WooCommerce and Elementor Pro to be activated.
- Dynamic tags will display fallback placeholders if viewed on a page or post that does not correspond to a valid WooCommerce product object.
