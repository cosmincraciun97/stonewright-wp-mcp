## Purpose
After choosing your images, click on the Create a new gallery button. On the next screen, if you want, add captions to your images, and then click Insert gallery.

## Use this when
- All available widgets are displayed
- Click or drag the widget to the canvas
- For more information, see Add elements to a page
- What is the Basic Gallery widget

## Settings highlights
- To access and use a widget – In Elementor Editor, click +. All available widgets are displayed.Click or drag the widget to the canvas. For more information, see Add elements to a page.
- Add a Basic Gallery widget – Step-by-step
- Select a link option from the Link field – link images to their Media Files, Attachment Pages, or choose None.
- Note – If you’re using the Lightbox feature, ensure the option Media Files is selected.
- Note – For images to display in the lightbox, you must link them to Media Files.
- Use the dropdown to choose – None: No caption is displayed.Attachment Caption: The image will be captioned as it is in the Media Library.
- Note – The lightbox will only work if you link the images to Media Files.
- Set the caption’s position – left, right, center, or justified.

## Limits / gotchas
- If you’re using the Lightbox feature, ensure the option Media Files is selected.
- For images to display in the lightbox, you must link them to Media Files.
- The lightbox will only work if you link the images to Media Files.
