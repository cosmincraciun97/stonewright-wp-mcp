## Purpose
The Basic Gallery widget displays a collection of images in a grid layout with lightbox functionality, enabling visitors to view full-size versions of photos by clicking thumbnails.

## Use this when
- You need to showcase a portfolio, product gallery, or image collection
- You want visitors to preview images without leaving the page
- Creating a simple, lightweight gallery without advanced filtering features
- Building portfolio sections, team photo galleries, or event image displays
- You prefer basic gallery functionality over the advanced Gallery widget

## Settings highlights
- **Add Images**: Upload or select multiple images from media library for the gallery
- **Gallery Layout**: Configure grid columns and spacing between image tiles
- **Lightbox Settings**: Enable/disable lightbox, control overlay appearance and animations
- **Image Sizing**: Set thumbnail dimensions and aspect ratio handling
- **Captions**: Add and display image titles or descriptions beneath thumbnails
- **Link Options**: Configure whether images open in lightbox or link to custom URLs
- **Hover Effects**: Apply zoom, fade, or other effects on image hover
- **Responsive Behavior**: Adjust column count for different device sizes

## Limits / gotchas
- Basic Gallery has fewer customization options compared to the advanced Gallery widget
- Lightbox functionality requires JavaScript enabled in visitor browsers
- Large image collections may impact page load performance without optimization
- Limited filtering and sorting capabilities compared to pro gallery solutions
