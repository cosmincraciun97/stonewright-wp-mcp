## Purpose
The Post Excerpt widget grabs and displays a short section of a post. It is generally used in an archive or Loop Grid.

## Use this when
- Create a Loop Grid and template
- The Post widget will now use the content of the post to create the post excerpt
- However, if there is a manually created post excerpt, the widget will use that post excerpt instead of taking content from the post
- Read the tutorial showing how to Create a Single Post Template

- Example – Adding a Post Excerpt to a Loop Grid
- Note – You can also include a post excerpt in a Heading or Text Editor widget by using a dynamic tag.

## Limits / gotchas
- You can also include a post excerpt in a Heading or Text Editor widget by using a dynamic tag.
