## Purpose
Most elements do not have visible borders by default. Use Border Type to create a visible border around the element. These borders help the element stand out and differentiate it from other elements.

## Use this when
- Most elements do not have visible borders by default
- Use Border Type to create a visible border around the element
- These borders help the element stand out and differentiate it from other elements
- The border radius tool allows you to adjust the shape of elements by rounding their corners
