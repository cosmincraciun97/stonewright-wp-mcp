## Purpose
The Breadcrumbs widget displays hierarchical navigation paths showing users' location within site structure. It helps visitors understand their position in the website hierarchy and provides easy navigation back to parent pages or sections.

## Use this when
- Building e-commerce sites where product category navigation is important
- Creating documentation or knowledge base structures with multiple levels
- Designing blog archives with category/tag hierarchies
- Improving SEO by showing content relationships to search engines
- Enhancing user experience on sites with deep page structures

## Settings highlights
- **Display separator** – Choose the character dividing breadcrumb items (slash, arrow, pipe)
- **Schema markup** – Enable structured data for search engine optimization
- **HTML tag selection** – Configure semantic HTML elements for accessibility
- **Text styling** – Adjust font size, color, weight, and family across breadcrumb items
- **Spacing controls** – Set padding and margins between breadcrumb elements
- **Link styling** – Customize colors and hover states for clickable breadcrumb links
- **Active item styling** – Style the current page differently from parent links
- **Responsive options** – Hide or adjust breadcrumbs on mobile devices

## Limits / gotchas
- Breadcrumbs widget is a Pro feature requiring Elementor Pro subscription
- Homepage typically doesn't display breadcrumbs since it's the navigation origin point
- Dynamic generation depends on proper WordPress page hierarchy and menu structure
- May require custom CSS adjustments for complex custom post types or taxonomies
