## Purpose
The Carousel widget enables you to display multiple items in a rotating, slide-based format. It automatically cycles through content or allows manual navigation, making it ideal for showcasing portfolios, testimonials, images, or products in an organized, space-efficient manner on your Elementor pages.

## Use this when
- You need to present multiple items without consuming excessive page real estate
- Building testimonial sections, image galleries, or product showcases
- Creating a rotating portfolio or case study display
- Displaying media that benefits from sequential, controlled viewing
- You want to encourage visitor engagement through interactive navigation

## Settings highlights
- Autoplay — configure automatic rotation timing and behavior
- Navigation arrows — toggle and style previous/next controls
- Pagination dots — enable slide indicators for quick navigation
- Slide speed — adjust animation duration between transitions
- Loop option — determine if carousel repeats after final slide
- Columns/items — control how many slides display simultaneously
- Spacing — manage gaps between carousel items
- Responsive settings — adjust column counts per device breakpoint
- Transition effects — select animation style (fade, slide, etc.)

## Limits / gotchas
- Performance may degrade with excessive slides or heavy media content
- Mobile responsiveness requires careful column configuration to avoid layout breaking
- Autoplay can impact accessibility; include manual controls and allow pausing
- Touch/swipe gestures have limited browser support across older devices
