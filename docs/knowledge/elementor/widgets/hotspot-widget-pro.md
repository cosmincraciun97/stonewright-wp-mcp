## Purpose
The Hotspot Widget focuses on helping you create interactive images that can help you highlight relevant information, boost engagement and increase conversions for your website.

## Use this when
- Offset – Use the slider to adjust the horizontal position of the hotspot in %
- Offset – Use the slider to adjust the vertical position of the hotspot in %
- Color – Use the color picker to choose your hotspot color
- Box Color – Use the color picker to choose the label background color

## Settings highlights
- Tip – You can use the Dynamic Options in the Link to open an Elementor Popup rather than a tooltip.
- Note – If the Custom Tooltip Position is enabled, the Custom Box Position will be used, rather than the general Tooltip Position.
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- If the Custom Tooltip Position is enabled, the Custom Box Position will be used, rather than the general Tooltip Position.
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
