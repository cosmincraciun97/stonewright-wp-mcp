## Purpose
Toggle to enable or disable the option to use schema. Style tab Accordion Accordion

## Use this when
- If you are using nested elements, the Accordion widget can include nested elements
- For details see, Add an Accordion widget below
- With the Toggle widget, however, all items are collapsed when a page is first loaded
- Only one item of an Accordion can be expanded at a time, automatically collapsing the previously opened item

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
