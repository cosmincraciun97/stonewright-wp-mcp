## Purpose
Add your Facebook Page Feed (previously named Like box) to your site.

## Use this when
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.
- Customizing specific styles, responsiveness, and display logic for elements across devices.

## Settings highlights
- URL – Type your Facbook Page URL
- Layout – Choose between Timeline, Events and Messages
- Small header – Displays a Facebook Header in a narrow view
- Cover – Show / hide the cover image
- Profile photos – Show photos of people who liked your page
- Custom CTA button – Choose between the page’s official CTA button, or the default Share Button
- Height – Set the height of the box (desktop, tablet and mobile)
- Note – If you are developing for European based sites, users must accept cookies, and be currently logged in to Facebook before the like, share, or comment buttons are displayed on the front end.

## Limits / gotchas
- If you are developing for European based sites, users must accept cookies, and be currently logged in to Facebook before the like, share, or comment buttons are displayed on the front end.
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
