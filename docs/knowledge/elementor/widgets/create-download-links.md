## Purpose
Enable website visitors to download files directly from your pages by creating clickable download links. This feature allows you to offer resources like documents, guides, or media files without requiring external hosting or complex configurations.

## Use this when
- You need to provide PDFs, documents, or other files for download
- Building resource libraries or documentation centers
- Creating gated content experiences
- Offering product manuals or guides to customers
- Setting up file distribution for multiple file types

## Settings highlights
- Link URL field accepts file paths from media library or external sources
- Download attribute forces file download instead of opening in browser
- Target settings control whether link opens in new window or current tab
- Anchor text customization for descriptive link labels
- Icon selection to visually indicate downloadable content
- Styling options for button appearance and hover effects
- Custom CSS classes for advanced design control

## Limits / gotchas
- Server configuration may restrict certain file types from downloading
- Large files may impact page performance or storage limits
- Browser security policies can override download behavior in some cases
- File accessibility depends on proper permissions and media library organization
