## Purpose
Configures WooCommerce Product Archive templates using Elementor Theme Builder to design standardized, dynamic category, tags, and shop index listing pages.

## Use this when
- Creating a cohesive grid style for all WooCommerce category/shop archive pages.
- Customizing archive headers, description areas, and product sorting queries globally.

## Settings highlights
- Templates > Theme Builder (navigation in WP Admin)
- Product Archive site part (left-hand sidebar)
- Add New archive template button
- Archive Products widget
- Query settings (source: current query, categories, tags)
- Display Conditions settings (apply to shop, product categories, search results)

## Limits / gotchas
- Requires WooCommerce and Elementor Pro to operate correctly.
- Only one product archive template can cover a single category at any given time; overlapping display conditions will cause unexpected template overrides.
