## Purpose
Background slideshow enables animated background image transitions on sections or containers. This feature cycles through multiple images with customizable timing and effects, creating dynamic visual depth without requiring video files or additional plugins.

## Use this when
- You want to add visual interest to hero sections or landing pages
- Creating photo galleries that auto-rotate in the background
- Building immersive homepage experiences with minimal performance impact
- You need subtle animations that don't distract from foreground content
- Designing portfolio or showcase sections with rotating imagery

## Settings highlights
- **Images**: Upload or select multiple images that will cycle through the slideshow
- **Transition speed**: Control how quickly each image fades or transitions to the next
- **Loop**: Enable continuous cycling or stop after final image
- **Transition effect**: Choose fade, slide, or other directional animations
- **Duration per slide**: Set how long each image displays before transitioning
- **Hover behavior**: Pause slideshow on user interaction or continue automatically
- **Overlay options**: Add color overlays or gradients to improve text readability
- **Lazy loading**: Optimize performance by loading images on-demand

## Limits / gotchas
- Large image files or many slides can impact page load times; optimize images before upload
- Autoplay slideshows may reduce accessibility — consider providing manual controls
- Background slideshows may not display consistently on older browsers or mobile devices with limited resources
