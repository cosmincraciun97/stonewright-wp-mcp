## Purpose
The Animated Headline widget creates dynamic text effects that cycle through different headline options with smooth animations, helping you draw visitor attention and emphasize key messaging on your pages.

## Use this when
- You need to highlight multiple value propositions or features in a rotating format
- Creating hero sections or call-to-action areas that demand visual focus
- You want to test different headline messaging without redesigning layout
- Building landing pages where animated text increases engagement metrics
- Emphasizing changing statistics, testimonials, or rotating content blocks

## Settings highlights
- **Headline text** – Enter multiple lines of text that will animate between each option
- **Animation type** – Choose effect style (clip, fade, scale, rotate, or typewriter animations)
- **Animation speed** – Control how quickly transitions occur between headlines
- **Loop settings** – Configure whether animation repeats infinitely or stops after cycles
- **Timing controls** – Adjust pause duration between headline changes
- **Typography styling** – Customize font size, color, family, weight, and letter spacing
- **Animation direction** – Determine whether text flows left-to-right or other directional effects

## Limits / gotchas
- Complex animations may impact page performance on lower-end devices; test mobile rendering
- Text length variations between headlines can cause layout shift; set fixed width containers
- Some animation types work better with shorter text; longer headlines may appear truncated
