## Purpose
Dynamic shortcodes allow you to embed WordPress shortcodes directly within Elementor pages and posts, enabling integration of plugin functionality and custom code output into your visual designs without manual coding.

## Use this when
- You need to display output from third-party plugins that use shortcodes
- Integrating custom shortcode functionality from your theme or custom plugins
- Adding dynamic content that updates based on shortcode parameters
- Combining plugin-generated content with Elementor's visual design tools
- You want to avoid building duplicate functionality within Elementor widgets

## Settings highlights
- Shortcode widget available in the Elementor editor for inserting shortcodes
- Support for standard WordPress shortcodes and custom plugin shortcodes
- Dynamic tag integration to make shortcodes data-driven
- Ability to pass parameters and attributes to shortcodes for customization
- Works alongside Elementor's native widgets and elements
- Compatible with both Elementor free and Pro versions
- Preview functionality to test shortcode output before publishing

## Limits / gotchas
- Some poorly-coded plugins may output styling that conflicts with Elementor's design
- Shortcode output cannot be styled directly through Elementor's style controls
- Performance may suffer if shortcodes trigger heavy database queries or external requests
- Not all third-party shortcodes are guaranteed to work flawlessly with Elementor's editor environment
