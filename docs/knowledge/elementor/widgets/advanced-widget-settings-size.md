## Purpose
The Size option in Elementor uses the Flex CSS properties of flex-grow and flex-shrink. The flex grow/shrink property specifies how the item will behave relative to the rest of the flexible items inside the same container when the window is resized.

## Use this when
- If you wish to use the Auto Size property, select the Grow or Shrink icon in the editor panel
- This automatically applies the property to best suit your layout
- If the Grow property is used, the item will attempt to use the remaining space in the container by growing to fit
- The default value is 1
