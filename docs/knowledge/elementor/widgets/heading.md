## Purpose
The Heading widget enables creation of stylized title headings throughout WordPress sites. It provides formatting controls for typography, size, color, and styling to make prominent text elements that organize page hierarchy and content structure.

## Use this when
- Creating page titles, section headers, or content divisions
- Emphasizing important text with custom font styling
- Establishing visual hierarchy within page layouts
- Building branded heading designs consistent with site identity
- Combining headings with other widgets for structured content sections

## Settings highlights
- **HTML Tag selection** — Choose heading levels (H1-H6) for semantic structure
- **Text content field** — Enter and edit heading copy directly
- **Typography controls** — Modify font family, size, weight, and style
- **Color options** — Set text color with global color integration
- **Alignment controls** — Position heading left, center, or right
- **Text effects** — Apply shadows, text stroke, or custom effects
- **Responsive sizing** — Adjust heading size per device breakpoint
- **Advanced styling** — Add padding, margins, borders, and backgrounds

## Limits / gotchas
- Heading widgets display as block-level elements requiring intentional spacing management
- Excessive nesting of multiple heading levels may confuse page structure and accessibility
- Custom font sizes on mobile devices require responsive rule configuration to prevent layout breaking
