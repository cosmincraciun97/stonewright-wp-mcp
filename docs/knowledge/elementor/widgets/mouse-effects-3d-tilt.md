## Purpose
Mouse Effects - 3D Tilt creates an interactive three-dimensional tilting animation on page elements that responds to cursor movement. The effect makes elements appear to rotate and shift in 3D space as visitors move their mouse across them, adding visual depth and engaging interactivity to static content — a premium feel for cards, images, and product showcases.

## Use this when
- Drawing attention to featured images, product showcases, or portfolio pieces
- Creating premium, interactive landing pages or hero sections
- Designing card-based layouts that benefit from dynamic hover depth
- Building ecommerce product displays that need enhanced visual appeal
- Developing portfolio or case study pages with gallery elements

## Settings highlights
- **3D Tilt**: enable/disable toggle under Advanced > Motion Effects > Mouse Effects
- **Intensity**: tilt angle range (lower = subtle, higher = dramatic rotation)
- **Speed**: how quickly the element responds to cursor movement
- **Scale**: optional scale-up transform applied during tilt
- **Perspective**: CSS perspective depth value controlling how pronounced the 3D effect appears
- **Max Tilt**: maximum degrees of rotation on X and Y axes
- **Gloss / Shine overlay**: optional highlight shimmer layer on tilt
- **Reset on Leave**: element snaps back to flat when cursor exits
- **Breakpoint disable**: per-device toggle to disable on mobile/tablet

## Limits / gotchas
- Requires Elementor Pro
- Heavy performance impact on mobile devices with limited GPU; disable on mobile breakpoints
- Excessive tilt values (over 20°) create disorienting UX — keep between 5–15° for usability
- Effect requires JavaScript enabled; degrades gracefully to static element without JS
- Multiple tilting elements on the same viewport simultaneously can strain older browsers
