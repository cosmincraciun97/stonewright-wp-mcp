## Purpose
Determines the spacing between items in the container and the items’ distance from the container’s border. Align Title

## Use this when
- In Elementor Editor, click +
- All available widgets are displayed
- Click or drag the widget to the canvas
- For more information, see Add elements to a page

## Settings highlights
- Choose a color – either use the color picker or a global color.The titles’ color can have three different styles:Normal – The default style.Hover – The style when users mouse over the tab.Active – The style when the tab is open.
- Choose an icon color – either use the color picker or a global color.You get three styles to choose for:Normal – The default style.Hover – The style when users mouse over the tab.Active – The style when the tab is open.
- Content options – Configure general content, title, tags, and icons.
- Style settings – Customize colors, borders, background, padding, and typography.
- Advanced features – Apply custom CSS classes, ID, and responsiveness properties.

## Limits / gotchas
- Set the direction of the tabs to either column or row.Note: If you set the direction to row, a slider appears, allowing you to adjust the width of the tabs in either PX or %.
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
