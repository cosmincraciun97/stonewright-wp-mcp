## Purpose
The Counter widget displays animated numerical statistics that increment from zero to a target number, creating visual emphasis for key metrics like client testimonials, project counts, or business achievements on your website.

## Use this when
- Showcasing business statistics or achievement milestones
- Highlighting numerical accomplishments (years in business, projects completed)
- Drawing attention to important performance metrics
- Creating engaging visual elements that animate on page load or scroll
- Building trust through quantifiable business results

## Settings highlights
- Target number field to set the final counted value
- Animation duration controls for speed of increment
- Prefix and suffix text options (e.g., "$" or "+")
- Number formatting options (decimals, thousands separator)
- Starting number configuration (typically zero)
- Text color, font size, and typography styling
- Alignment options for layout positioning
- Trigger settings for when animation begins (on load or on scroll)

## Limits / gotchas
- Animation only triggers once per page load unless refresh occurs
- Very large numbers may display slowly depending on animation duration settings
- Mobile responsiveness requires font size adjustment to prevent text overflow
- Counter animation may not be visible to users with reduced motion preferences enabled
