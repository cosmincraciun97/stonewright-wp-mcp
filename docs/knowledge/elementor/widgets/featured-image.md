## Purpose
Displays a post or page's featured image within Elementor layouts. This widget automatically pulls the assigned featured image from WordPress post/page settings, allowing builders to control placement, sizing, and styling independently from the theme's default featured image display.

## Use this when
- Building custom single post or page templates
- Creating portfolio or case study layouts with hero images
- Designing archive pages with prominent image displays
- Need independent control over featured image styling
- Building WooCommerce product templates

## Settings highlights
- Image source selection (featured image or custom)
- Width and height customization
- Aspect ratio controls
- Link options (to post, custom URL, or lightbox)
- Image fit settings (fill, contain, cover)
- Border and shadow effects
- Responsive image sizing
- Overlay options for text/effects

## Limits / gotchas
- Requires a featured image assigned to the post/page to display content
- Image dimensions depend on what WordPress has generated in media library
- Lightbox functionality requires separate lightbox configuration
- Performance impacts with large unoptimized featured images across multiple widgets
