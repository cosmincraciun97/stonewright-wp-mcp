## Purpose
Hover animations are interactive effects that activate when users place their cursor over elements on a webpage. These animations enhance user engagement by providing visual feedback and creating a more dynamic, polished browsing experience without requiring page navigation or complex interactions.

## Use this when
- You want to draw attention to interactive elements like buttons or links
- Creating visual feedback to indicate that an element is clickable
- Building more engaging and modern-looking web designs
- Enhancing user experience through subtle motion effects
- Adding polish to call-to-action sections or navigation menus

## Settings highlights
- Animation type selection from built-in library
- Duration controls for animation speed
- Delay settings to stagger effects
- Easing options for natural motion
- Trigger conditions (hover, click, or other interactions)
- Animation intensity or scale adjustments
- Color or opacity transitions
- Reset behavior after animation completes

## Limits / gotchas
- Hover animations don't function on touch devices without additional configuration
- Excessive animations can negatively impact page performance and load times
- Some users may have motion sensitivity or accessibility concerns with animations
- Mobile users won't see hover effects unless paired with alternative interaction triggers
