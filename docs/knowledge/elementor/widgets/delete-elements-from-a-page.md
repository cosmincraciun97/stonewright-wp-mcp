## Purpose
Delete elements like widgets, sections, and columns from your Elementor page to remove unwanted content and clean up your design layout during the editing process.

## Use this when
- You want to remove a widget or section that's no longer needed
- Cleaning up page structure after reorganizing content
- Deleting entire sections, columns, or nested elements
- Removing test content before publishing
- Simplifying page layouts

## Settings highlights
- Select the element directly on the canvas to reveal deletion options
- Right-click context menu provides "Delete" action for quick removal
- Delete icon typically appears in element toolbar or panel controls
- Applies to all element types: widgets, sections, columns, containers, inner sections
- Works with nested elements within accordion, tabs, and toggle widgets
- Action is immediate and affects page structure instantly

## Limits / gotchas
- Deleted elements cannot be recovered unless you use the revision history or undo function
- Deleting a section removes all contained columns and widgets simultaneously
- No confirmation dialog appears — deletion executes immediately upon selection
- Changes must be saved to persist; use preview mode to verify before publishing
