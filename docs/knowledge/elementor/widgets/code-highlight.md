## Purpose
The Code Highlight widget displays formatted code snippets with syntax highlighting for better readability. It is designed to showcase code samples on web pages with proper formatting and visual distinction of different code elements, making it ideal for developer documentation or technical blogs.

## Use this when
- Displaying code examples or tutorials on your website
- Creating documentation pages that reference code snippets
- Teaching programming concepts with visual code samples
- Building developer resource pages or technical blogs
- Showcasing API examples or configuration code

## Settings highlights
- Language selection for syntax highlighting (multiple languages supported)
- Code input area for pasting or writing code
- Theme options for code appearance
- Line numbers toggle
- Copy-to-clipboard functionality
- Font size and family customization
- Background color options
- Border and padding controls
- Responsive text scaling

## Limits / gotchas
- Very long code blocks may require scrolling on mobile devices
- Some programming languages may have limited syntax highlighting support
- Performance may be affected with multiple large code blocks on one page
- The widget requires proper code formatting for optimal display
