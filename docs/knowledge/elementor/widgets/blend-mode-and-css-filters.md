## Purpose
Blend modes and CSS filters are visual effects tools that allow you to control how elements interact with layers beneath them and apply photographic-style adjustments. These features enable sophisticated visual styling without requiring image editing software.

## Use this when
- You need to create layered visual effects where elements blend with background content
- Applying filters like blur, brightness, contrast, or saturation to images or sections
- Designing overlay effects for text over images or darkening/lightening backgrounds
- Creating sophisticated color grading or transparency effects
- Building professional-looking designs with non-destructive visual adjustments

## Settings highlights
- **Blend Mode dropdown**: Select from modes like Multiply, Screen, Overlay to control element layering behavior
- **Opacity control**: Adjust transparency levels for blend mode effects (0-100%)
- **Blur filter**: Apply Gaussian blur with adjustable pixel radius
- **Brightness/Contrast**: Fine-tune exposure and tonal range independently
- **Saturation control**: Increase or decrease color intensity from grayscale to oversaturated
- **Hue Rotation**: Shift entire color spectrum by degrees (0-360°)
- **Grayscale toggle**: Convert elements to black and white with percentage control
- **Invert colors**: Reverse color values for artistic effects
- **Sepia filter**: Apply warm, vintage-looking tone overlay
- **Filters stack**: Apply multiple filters simultaneously for combined effects

## Limits / gotchas
- Browser support varies for certain blend modes and filters on older versions
- Heavy filter stacking may impact page performance, especially on mobile devices
- Filters apply to the entire element including children; cannot target specific child elements selectively
- Some filters cannot be animated smoothly across all browsers despite CSS support
