## Purpose
This feature allows you to build interactive audio playlists directly within Elementor pages. Users can organize multiple audio tracks and present them with playback controls, enabling seamless audio content delivery on your website without relying on external embedding services.

## Use this when
- You want to showcase multiple music tracks or audio files on a single page
- Building a music portfolio, podcast archive, or audio library section
- Creating dedicated listening experiences for audio content
- You need native audio playback with playlist management capabilities
- Designing musician/artist portfolio or audio creator websites

## Settings highlights
- Add multiple audio tracks to a single playlist widget
- Configure playback controls (play, pause, volume, progress bar)
- Set custom playlist titles and track metadata
- Choose between different playlist layout display options
- Control autoplay behavior and looping settings
- Customize player appearance to match your design
- Add cover art or featured images for tracks
- Enable/disable shuffle and repeat functionality

## Limits / gotchas
- Audio file format compatibility depends on browser support (MP3, WAV, OGG)
- Large audio files may impact page load performance; optimize before uploading
- Mobile responsiveness may require additional styling adjustments for smaller screens
- Some older browsers may not support all playlist widget features
