## Purpose
Link In Bio widgets are a purpose-built set of Elementor elements for creating a mobile-optimized "link in bio" page — a single-page micro-site that aggregates multiple links, social profiles, and media in a vertical scroll layout. They address the common influencer/creator pattern of directing social traffic to a curated hub of links.

## Use this when
- Building an Instagram / TikTok / Twitter link-in-bio landing page
- Creating a personal brand hub with social links, latest content, and contact CTA
- Designing a simple mobile-first page aggregating multiple URLs for a creator or small business
- Replacing third-party link-in-bio SaaS tools (Linktree, etc.) with a self-hosted WordPress version

## Settings highlights
- **Avatar / Profile Image**: circular or square profile photo at the top of the page
- **Display Name / Bio**: headline and short description text widgets
- **Link Button rows**: styled button widgets with icon, label, and URL for each link
- **Social Icons bar**: compact horizontal icon strip for social profile links
- **Media embeds**: YouTube, Spotify, and other media preview blocks
- **Background options**: gradient, solid, or image backgrounds for the whole page
- **Responsive layout**: single-column stacked layout optimized for mobile viewports
- **Custom Fonts / Colors**: per-element typography and color controls consistent with site kit

## Limits / gotchas
- No dedicated "Link In Bio" widget panel — these are standard Elementor widgets arranged in the Link In Bio page template pattern
- Analytics on individual link clicks requires a third-party link-tracking plugin or UTM parameters per link
- Does not auto-sync with social media platforms; content must be updated manually
- Canvas page template (no header/footer) is recommended for a clean link-in-bio experience
