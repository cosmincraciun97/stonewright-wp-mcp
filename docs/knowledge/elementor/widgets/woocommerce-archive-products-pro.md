## Purpose
Configures the Archive Products widget styles and query parameters for WooCommerce listing pages.

## Use this when
- Formatting lists of items within a Product Archive template.
- Controlling the number of columns, order, and pagination styles for shop products.

## Settings highlights
- Allow Order dropdown (Yes/No toggle)
- Columns count setting (1 to 12)
- Rows count setting
- Query settings (current query, etc.)
- Pagination dropdown (numbers, prev/next, none)
- Product Image visibility toggle
- Product Title visibility toggle
- Product Price visibility toggle
- Add to Cart visibility toggle
- Rating and Sale Badge visibility toggles

## Limits / gotchas
- Only works within an active WooCommerce store context.
- Overlapping global archive conditions can cause rendering conflicts.
