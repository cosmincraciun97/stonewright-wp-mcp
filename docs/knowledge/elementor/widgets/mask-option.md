## Purpose
Masks allow you to control which parts of an element are visible by applying a shape or image as an overlay. This design technique lets you crop content into custom shapes, create artistic effects, and enhance visual hierarchy without modifying the underlying element.

## Use this when
- You want to display images or content in non-rectangular shapes
- Creating decorative effects with gradients or custom shapes
- Highlighting specific portions of an image while hiding the rest
- Building branded design elements with unique silhouettes
- Combining multiple visual layers for sophisticated compositions

## Settings highlights
- Mask field — primary control for applying mask settings to elements
- Shape options — select from predefined masks or upload custom images
- Visibility control — determines which areas of the element display or hide
- Opacity adjustment — fine-tune transparency of masked regions
- Responsive behavior — masks adapt across different device breakpoints
- Layer blending — combine masks with blend modes for enhanced effects

## Limits / gotchas
- Mask application depends on browser support for CSS mask properties
- Performance may impact on elements with complex mask shapes
- Masks don't affect the underlying element's actual dimensions or click zones
