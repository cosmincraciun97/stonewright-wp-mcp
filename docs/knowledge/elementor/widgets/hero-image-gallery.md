## Purpose
Build an image gallery displayed as a hero section using Elementor's container system. This layout showcases multiple images prominently at the top of a page, combining visual impact with organized image presentation for portfolio, product, or showcase-style content.

## Use this when
- Displaying a collection of high-impact images at page entry
- Creating portfolio or showcase sections with multiple photos
- Building image-heavy landing pages that lead with visual content
- Organizing product galleries or service showcases
- Featuring before/after or step-by-step visual narratives

## Settings highlights
- Container layout configured with flexbox or grid arrangement
- Background image options for each gallery item
- Image sizing and scaling controls for responsive display
- Hover effects and transitions for interactive engagement
- Spacing and padding adjustments between gallery elements
- Overlay text or captions positioned over images
- Lightbox settings for full-screen image viewing
- Mobile responsiveness breakpoints for different devices

## Limits / gotchas
- Large image files may impact page load performance — optimize images before uploading
- Grid layouts require careful planning across mobile, tablet, and desktop viewports to maintain visual hierarchy
- Text overlays need sufficient contrast against background images for accessibility compliance
