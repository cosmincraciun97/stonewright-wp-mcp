## Purpose
The Author Box widget displays information about a post's author, including their name, bio, avatar, and social links. This element helps readers learn more about content creators and builds author credibility on blog posts and articles.

## Use this when
- You want to feature author credentials below or alongside blog post content
- Creating single post templates that need author attribution
- Building community-focused sites where author identity matters
- Designing layouts that encourage reader connection with writers
- Setting up multi-author blogs with consistent author presentation

## Settings highlights
- **Author selection**: Choose which author profile to display
- **Avatar display**: Toggle and customize author profile image size
- **Bio text**: Show or hide author biography section
- **Social icons**: Link to author's social media profiles
- **Custom links**: Add author website or portfolio URLs
- **Alignment options**: Position author box left, center, or right
- **Styling controls**: Adjust typography, spacing, background colors
- **Hover effects**: Add interactive animations to the widget

## Limits / gotchas
- Author box pulls from WordPress user profiles — incomplete user data won't display properly
- Social icons require manual configuration in author profile settings
- Widget only shows one author at a time; use multiple instances for co-authored posts
- Responsive behavior may require custom styling on mobile devices
