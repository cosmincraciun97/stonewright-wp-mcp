## Purpose
The Divider Widget allows you to add styled horizontal lines that divide your content.

## Use this when
- Organizing your layout design and structuring content elements inside Elementor.
- Enhancing user experience by presenting information in a clean, professional, and accessible layout.
- Customizing specific styles, responsiveness, and display logic for elements across devices.

## Settings highlights
- Style – Choose between many classic and patterned styles, such as solid, curly, wavy, tribal patterns, arrows, pluses and more.
- Width – Control the width of the divider as percentage from 0 to 100 percent.
- Alignment – Align the divider to the left, center or right of the page.
- Add Element – Select from None, Text, or Icon. Select Text or Icon allows you to either enter the Text to be included or select or upload an icon from the Icon Library.
- Color – Choose the color of the divider
- Size – Set the size/height of the divider, from 1 to 100, in either pixels or as a percentage
- Amount – Set the number of patterned elements to show
- Gap – Slide to set the gap above and below the divider, from 1 to 50

## Limits / gotchas
- Prerequisites: Ensure you are using the correct Elementor and Elementor Pro versions compatible with this feature.
- Performance: Having too many nested elements or widgets on a single page can affect site speed and core web vitals.
