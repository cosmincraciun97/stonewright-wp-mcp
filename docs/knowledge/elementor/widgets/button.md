## Purpose
The Button Widget allows designers to create customizable, clickable buttons without requiring additional plugins or shortcodes. This versatile element functions as a core interaction component for directing user actions toward specific destinations or triggering defined behaviors on web pages.

## Use this when
- Adding call-to-action elements that guide visitors toward conversions
- Creating navigation links with enhanced visual styling and interactivity
- Building form submission triggers or download prompts
- Designing multi-button layouts for product selections or feature highlights
- Establishing consistent clickable elements throughout page designs

## Settings highlights
- Link URL configuration for external sites, internal pages, or anchor links
- Button text customization with full typography controls (font, size, weight, color)
- Background color, gradient, and hover state styling options
- Padding and margin adjustments for spacing control
- Border radius settings for rounded or sharp corner styles
- Icon placement options (before/after text or standalone)
- Size presets (small, medium, large) or custom dimensions
- Advanced styling including shadow, opacity, and transform effects

## Limits / gotchas
- Button styling is limited to predefined widget settings; complex custom layouts require custom CSS
- Icon display depends on proper icon library integration and selection
- Link behavior respects standard HTML anchor attributes; dynamic redirects need form actions configured separately
