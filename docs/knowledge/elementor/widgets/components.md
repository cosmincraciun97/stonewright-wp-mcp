## Purpose
Components in Elementor are reusable building blocks that allow you to create consistent, modular design elements across your website. They enable you to build once and apply the same styled element multiple times, with synchronized updates across all instances, reducing redundant design work and ensuring site-wide consistency.

## Use this when
- You need to reuse the same design element across multiple pages
- You want to maintain design consistency throughout your site
- You need to make global updates to repeated elements simultaneously
- You're building a design system for your website
- You want to reduce redundant design work

## Settings highlights
- Create components from existing page elements
- Edit component instances individually or globally
- Sync updates across all component uses
- Organize components in your library
- Set component-specific styling and behavior
- Control which properties update globally vs. locally
- Nest components within other components
- Apply responsive settings to components

## Limits / gotchas
- Components are site-specific and cannot be transferred between websites like templates
- Editing a component globally affects all instances; use caution with bulk updates
- Component library has storage limitations based on your Elementor plan
- Some advanced customizations may not translate properly across all component instances
